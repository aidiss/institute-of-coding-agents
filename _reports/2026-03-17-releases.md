# Releases — 2026-03-17

## Models

### NVIDIA Nemotron 3 Super — NVIDIA (Mar 11, GA)
Announced at GTC 2026 on March 11. A 120B-total-parameter hybrid Mixture-of-Experts model with only 12B active parameters per forward pass. Designed for complex multi-agent applications including software development, cybersecurity triaging, and agentic workflows.

- **60.47% on SWE-Bench Verified** — strong coding benchmark result for an open-weights model.
- **2.2× higher throughput** than GPT-OSS-120B.
- **1M-token context window**.
- Ships with open weights, datasets, and full training recipe under the NVIDIA Nemotron Open Model License.

Available now via NVIDIA NIM and compatible inference runtimes.

---

### Gemini Embedding 2 — Google (Mar 10, Public Preview)
Google's first fully multimodal embedding model. Available in public preview via the Gemini API and Vertex AI as `gemini-embedding-2-preview`.

- Maps **text, images, video, audio, and documents** into a single unified embedding space across 100+ languages.
- Supports **interleaved multimodal input** in a single request.
- Implements **Matryoshka Representation Learning (MRL)** — most critical semantic information packed into earliest dimensions; vectors can be safely truncated to 768, 1536, or 3072 dimensions.
- Context: up to 8,192 input tokens (text), 80s audio, 128s video.
- Early adopters report latency reductions up to **70%** and near-double semantic similarity scores in production RAG pipelines.

Integrations: LangChain, LlamaIndex, Haystack, Weaviate, QDrant, ChromaDB, Vertex Vector Search.

### IBM Granite 4.0 1B Speech — IBM (Mar ~17, GA)
Compact 1B-parameter speech model for multilingual ASR and bidirectional speech translation. New additions: Japanese language support and keyword biasing. Footprint under 1.5 GB VRAM — targets edge devices and low-resource servers.

### LTX 2.3 — Lightricks (early Mar, GA)
22B-parameter open-source video generation model. Generates 4K video at 50 FPS with synchronized audio in a single pass; supports portrait mode (1080×1920); runs in 8 denoising steps on the distilled variant. Open weights, commercially licensed.

No new developments on Mar 17, but enterprise rollout continues. Recap: 1.05M-token context, native computer use, 33% fewer factual errors than GPT-5.2, variants include Thinking and Pro. Pricing: $2.50/$15 per 1M tokens (standard), $30/$180 (Pro). GPT-5.1 retired Mar 11.

## Products

### NemoClaw — NVIDIA (Mar 16, GA at GTC)
NVIDIA's open-source enterprise agent platform, built to fill the gap after OpenAI acquired OpenClaw in February 2026. Key capabilities:

- Runs **OpenClaw-compatible agents** on-premises using RTX PRO 6000 Blackwell GPUs (up to 4,000 TOPS, 96 GB GPU memory).
- Includes NVIDIA OpenShell runtime — a secure, sandboxed environment for autonomous agents.
- Built-in RBAC, audit logs, and data residency controls for enterprise governance.
- Native multi-agent orchestration; partner integrations with Salesforce, Cisco, Google, Adobe, and CrowdStrike.

---

### NVIDIA Vera Rubin — NVIDIA (Mar 16, announced at GTC; production rollout later 2026)
NVIDIA's next-generation data center architecture unveiled at GTC 2026:

- Seven chips now in full production; NVL72 rack achieves **10× lower cost per token** at inference vs. Blackwell.
- New **Kyber** design integrates 144 GPUs in vertical compute trays for higher density — ships in Vera Rubin Ultra (2027).
- **Rubin CPX** variant specialized for massive-context tasks (1M-token code analysis, generative video).
- **10× more performance per watt** than Grace Blackwell.
- Azure is first hyperscale provider to power up NVL72 systems.
- Jensen Huang projects **$1 trillion in combined Blackwell + Vera Rubin purchase orders** through 2027.
- NVIDIA also announced Vera Rubin Space-1 — Vera Rubin-based orbital data centers; radiation hardening in development.
