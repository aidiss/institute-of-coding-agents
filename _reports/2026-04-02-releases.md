# Releases — 2026-04-02

## Models

### xAI — Grok 4.20

- **Grok 4.20** — GA (released February 2026). Novel parallel sub-agent architecture: four specialized sub-agents run concurrently on complex queries and merge results. Enhanced real-time web access. Available via X Premium at $50/month. Grok 5 targeting Q2 2026.

### Mistral — Mistral Small 4

- **Mistral Small 4** — GA (released March 3, 2026). 6B active parameters; incorporates Devstral's agentic coding capabilities into a compact model. Notable licensing shift: **Apache 2.0**, marking a significant move away from prior restrictive licensing. Open weights available on Hugging Face; also available via Mistral API.

### Anthropic — Claude Mythos (limited access)

- **Claude Mythos** (codename) — Training confirmed complete as of March 26, 2026. In limited early access with cybersecurity partners only. No public release date announced. Not yet available on consumer plans.

## Products

### OpenAI Codex

- **Windows app** — GA (launched March 4, 2026). Follows the macOS app (February 2, 2026). Recent April updates added: Windows sandbox proxy-only networking, ChatGPT device-code sign-in, and dynamic bearer tokens for custom model providers. CLI remains open-source (Rust). Included with ChatGPT plans (promotional period).
- **Codex Security** — Research preview. Application security review agent, operating as a specialized subagent of the Codex cloud runner.

### Devin (Cognition AI)

- **Devin 2.2** — GA. Key new feature: **Interactive Planning Checkpoints** — surfaces relevant files and a preliminary plan before consuming ACU credits, allowing the user to modify the plan before compute is spent. Reduces wasted cycles on misaligned tasks. Pricing: $20/month Core (down from $500/month at launch); team plans scale via ACU credits ($2.25/ACU, ~15 min of agent work).

### Augment Code — Intent

- **Intent** — Public beta (launched February 2026). Standalone macOS desktop app. Spec-driven multi-agent orchestration: multiple agents share a living specification document and coordinate around a single evolving plan. New **`emit_notification` MCP tool**: external services (CI, issue trackers, monitoring) can push notifications into a workspace to wake specific agents. Windows waitlist open; no GA pricing announced (credit-based model).

### JetBrains Central

- **JetBrains Central** — Announced March 24, 2026. A control layer connecting JetBrains Air IDE and Junie coding agent into a unified agentic platform. Design partner early access begins Q2 2026. Status: **not yet GA**.

### Google Gemini Code Assist

- **Free for individuals** — GA (March 2026). Full-featured access made completely free for individual developers (not a reduced tier). Deeper GCP stack integration: Cloud Run, BigQuery, and infrastructure code generation.

## Key Themes

- **Spec-driven development emerging as a pattern**: AWS Kiro, Augment Intent, and Devin 2.2 all center on generating or modifying a structured plan/spec before code generation — a distinct workflow layer above pure autocomplete or agentic chat.
- **Parallel agents now ubiquitous**: Cursor (8 agents), Windsurf Wave 13 (5 Cascade agents), Grok 4.20 (4 parallel sub-agents), Claude Code (Agent Teams), and Codex cloud all ship concurrent-agent capabilities.
- **Open-weight licensing liberalizes**: Mistral Small 4 under Apache 2.0 continues the trend of open-weight models adopting permissive licensing to drive ecosystem adoption.
- **Pricing pressure downward**: Devin dropped from $500/month to $20/month; Gemini Code Assist went fully free for individuals; Mistral Small 4 is open-weight. Established pricing floors are eroding.
- **DeepSeek V4 still pending**: Despite anticipation, no V4 release confirmed in this window; V3.2 (late 2025) remains current.
