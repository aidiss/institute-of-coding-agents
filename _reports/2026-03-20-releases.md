# Releases — 2026-03-20

What shipped this week (March 13–20, 2026) across models, products, and open-source tools in the AI coding agent landscape.

---

## Models

### Mistral Small 4
- **Released:** March 16, 2026
- **Availability:** GA, Apache 2.0
- 119B-parameter Mixture-of-Experts model with 6.5B active parameters. First Mistral model to unify reasoning (Magistral), multimodal (Pixtral), and agentic coding (Devstral) in a single checkpoint. 256K context window.
- **Pricing:** $0.15/M input, $0.60/M output
- **Available on:** Mistral API, Hugging Face, AI Studio, NVIDIA NIM (day-0)
- Strong SWE-bench performance at substantially lower cost than frontier models.

### NVIDIA Nemotron 3 + Isaac GR00T N1.7
- **Released:** March 16, 2026 (GTC, San Jose)
- **Availability:** GA for select variants
- Three-model family: Nemotron 3 Ultra (120B, 5× throughput on Blackwell via NVFP4), Nemotron 3 Omni (audio + vision + language), Nemotron 3 VoiceChat (real-time conversation). Isaac GR00T N1.7 is an open reasoning vision-language-action model for humanoid robots, now commercially viable.
- **Available on:** GitHub, Hugging Face, NVIDIA NIM
- Listed adopters: Cursor, CodeRabbit, Factory.

### GPT-5.4 Mini and Nano
- **Released:** Rolling out across Codex/API during the week of March 13–20
- **Availability:** GA
- Mini runs at 2× the speed of GPT-5.4 and costs 30% of a GPT-5.4 quota unit in Codex. Nano is the lowest-cost option in the family.
- **Pricing (Nano):** $0.20/M input, $1.25/M output
- Both available in the Codex app, CLI, IDE extension, and API.

## Products

### Claude Code v2.1.71–2.1.76
- **Released:** Multiple updates, March 11–15, 2026
- **Availability:** GA
- Key additions:
  - `/loop` command (v2.1.71): session-scoped task scheduler — runs a prompt on a user-specified interval (e.g., `/loop 5m check the deploy`).
  - Skills system: reusable instruction sets packaged in `.claude/skills/SKILL.md` files that Claude activates based on task context.
  - Opus 4.6 context window raised to 1M tokens (Max, Team, Enterprise plans) as of v2.1.75 (March 13).
  - Output token limits for Opus 4.6: 64K default, 128K upper bound.
  - Voice mode (`/voice`, push-to-talk via spacebar): progressive rollout.
- Bug fixes: 100% CPU loop on permission prompts, deadlock on skill-file changes during `git pull`.

### Anthropic Code Review
- **Released:** March 9, 2026
- **Availability:** Research Preview — Teams and Enterprise plans only (not Pro or Max)
- Multi-agent automated PR review. On PR open, parallel agents inspect logic errors, boundary conditions, API misuse, auth flaws, and project conventions; cross-verify findings; post a single overview comment and inline annotations.
- **Pricing:** ~$15–25 per review; ~20 minutes per run
- **Platforms:** GitHub only at launch (no GitLab, Azure DevOps, or Bitbucket)
- Internal adoption at Anthropic raised substantive review coverage from 16% to 54% of PRs.

### GitHub Copilot Coding Agent for Jira
- **Released:** March 5, 2026
- **Availability:** Public Preview
- Assign a Jira issue to Copilot; it writes code and opens a draft PR, posts status updates in Jira, and asks clarifying questions there. Follows the repo's existing PR review rules.
- **Pricing:** No additional cost beyond existing Copilot subscription
- Requires: Atlassian Forge app + GitHub app. Available on Atlassian Marketplace.

### Cursor Automations
- **Released:** Early March 2026
- **Availability:** GA for paying users
- Event-driven agent system triggered by codebase changes, Slack messages, or timers. Flagship use case: Bugbot runs on every new code push. Cursor also reports PagerDuty-triggered incident-response automations querying server logs via MCP.
- Cursor milestones this month: $2B annualized revenue, $29.3B valuation, JetBrains IDE support (removing VS Code lock-in).

### OpenAI Codex CLI (patch)
- **Released:** March 19, 2026
- **Availability:** GA (open source)
- Hardening release: fixed plugin feature gating, improved Linux sandbox startup on symlinked checkouts, polished A2A/realtime session startup.

### Gemini CLI v0.33.0
- **Released:** March 11, 2026
- **Availability:** GA, Apache 2.0
- HTTP authentication for A2A remote agents, authenticated A2A agent card discovery, Plan Mode enabled by default with built-in research subagents.
- **Pricing:** Free tier via personal Google account — Gemini 2.5 Pro at 60 req/min, 1,000/day.

### Kiro Students
- **Released:** March 18, 2026
- **Availability:** GA (new access tier)
- AWS's Kiro IDE (spec-driven, VS Code fork, Claude Sonnet 4.0/3.7) now offers a Students plan: free one-year access with 1,000 credits/month for college developers, then drops to the free tier (50 credits/month).

## New Entrants

### Entire / Checkpoints
- **Launched:** February 10, 2026 (most recent release)
- Founded by former GitHub CEO Thomas Dohmke. $60M seed round at $300M valuation (Felicis, Madrona, Microsoft M12, Basis Set).
- First product: **Checkpoints** — open-source CLI that auto-captures agent context (prompts, transcripts, files touched, token usage, tool calls) as versioned data in Git alongside each commit. Supports Claude Code and Gemini CLI at launch.
- Positioned as the provenance and audit layer for AI-generated code, not a competing IDE.

## Signals

- **Multi-agent is now baseline.** Every major tool shipped parallel agent support in late February (Windsurf 5 parallel agents, Claude Code 16+, Codex CLI subagent delegation, Grok Build 8 agents). Running simultaneous agents on different parts of a codebase is no longer a differentiator.
- **Code review as a product category.** Anthropic Code Review is the first standalone multi-agent PR review product from a major lab, introducing per-review pricing ($15–25) rather than seat licensing.
- **Provenance is emerging as a market.** Entire's Checkpoints and Claude Code's Skills system both address "why did the agent do this" — capturing agent decisions alongside code.
- **No new flagship model drop from OpenAI, Google, Anthropic, or Meta** in the 24 hours preceding March 19. The week was infrastructure and tooling, not frontier launches.
- **Windsurf SWE-1.5 free window closing.** Made free December 24, 2025 for three months — the free period expires around late March 2026.
