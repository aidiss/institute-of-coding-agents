## Products

### Claude Code — v2.1.74 (Anthropic)

- **Released March 12, 2026** — latest stable on npm/PyPI.
- **Actionable `/context` suggestions** — new diagnostics identify context-heavy tools, memory bloat, and capacity warnings with specific optimization tips.
- **`autoMemoryDirectory` setting** — configure a custom directory for auto-memory storage.
- **Memory leak fix** — streaming API response buffers were not released when the generator was terminated early, causing unbounded RSS growth on the Node.js/npm code path.
- **Security fix** — managed-policy `ask` rules could previously be bypassed by user `allow` rules or skill `allowed-tools`; now correctly enforced.
- **Bug fixes** — fixed full model IDs (e.g., `claude-opus-4-5`) being silently ignored in agent frontmatter; fixed marketplace update not syncing git submodules; fixed unknown slash commands with arguments silently dropping input (now shows a warning); fixed Hebrew, Arabic, and other RTL text not rendering in Windows Terminal / conhost / VS Code integrated terminal; fixed LSP servers not working on Windows due to malformed file URIs.
- **`--plugin-dir` change** — local dev copies now override installed marketplace plugins with the same name.
- **[VSCode]** Fixed delete button not working for Untitled sessions.

### Windsurf — v1.9577.27 (Cognition)

- **Released March 12, 2026** — patch release.
- **Apple M5 fix** — resolves a regression affecting Windsurf on the latest Apple Silicon hardware.
- Wave 13 multi-agent / Git worktrees features and the free SWE-1.5 model remain available through end of March 2026.
- Phoenix Alpha available in Windsurf Next (pre-release channel) since March 3.

### Cursor — ACP Integration in JetBrains IDEs

- **Released this week** — Cursor is now available inside **IntelliJ IDEA, PyCharm, WebStorm**, and other JetBrains IDEs via the **Agent Client Protocol (ACP)**.
- Provides secure codebase indexing, semantic search, and access to frontier models (OpenAI, Anthropic, Google, Cursor-native) from within JetBrains.
- Free for all users on paid Cursor plans; authenticate with an existing Cursor account.
- Background agents, Bugbot, and cloud automations (GA March 5) remain current in the standalone VS Code fork.

### ChatGPT — Interactive Visual Learning (OpenAI)

- **Rolled out March 10–13** to all logged-in users — math and science visual learning (previously in beta) now broadly available.
- Also shipping: an **experimental code mode** for isolated coding workflows and a **hooks engine** (SessionStart / Stop hook events).

### GitHub Copilot / VS Code

No new release. VS Code 1.109 multi-model parallel agents (Claude, Codex, and Copilot agents running concurrently, January 2026) fully rolled out.
