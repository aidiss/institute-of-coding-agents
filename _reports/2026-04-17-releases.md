# Releases — 2026-04-17

## Models

### Meta
- **Llama 4 Scout** — Released April 5, 2026 (open-weight, GA). 17B active parameters, 16 experts MoE, 10M-token context window. Natively multimodal (text, image, video). Outperforms Gemma 3, Gemini 2.0 Flash-Lite, and Mistral 3.1 on broad benchmarks. Available on Hugging Face and llama.com; powers Meta AI on WhatsApp, Messenger, and Instagram.
- **Llama 4 Maverick** — Released April 5, 2026 (open-weight, GA). 17B active parameters, 128 experts MoE. Beats GPT-4o and Gemini 2.0 Flash across multimodal benchmarks; comparable to DeepSeek v3 on reasoning/coding at less than half the active parameters. Open-weight download available.
- **Meta Muse Spark** — Released April 8, 2026 (GA). Meta's first proprietary frontier model — no open weights. Targets enterprise creative and productivity workloads.

### Google DeepMind
- **Gemma 4** — Released April 2, 2026 (open-source, Apache 2.0). Four variants: E2B, E4B, 26B MoE, 31B Dense. First fully open multimodal model from Google (text, image, audio, video); 256K context window. E2B/E4B optimised for edge devices (phones, Raspberry Pi, Jetson Orin Nano). Available on Hugging Face and Google Cloud.

## Products

### Anthropic — Claude Code
- **Desktop App Redesign** — Released April 14, 2026. Parallel sessions with drag-and-drop panes, integrated terminal. Available on Pro ($20/month), Max, Team, and Enterprise plans (not free tier).

### Anysphere — Cursor
- **Cursor 3 + Composer 2** — Released April 2026 (GA). Composer 2 is Anysphere's first in-house frontier coding model: 61.3 on CursorBench (+39% over Composer 1.5), 200+ tokens/second via custom GPU kernels, default model in Auto mode. Cursor 3 also ships cloud agents on isolated VMs, `/worktree` for isolated branch changes, self-hosted agents, and parallel Agent Tabs.

### Google
- **Gemini CLI** — Released April 2026 (open-source, Apache 2.0). Official Google terminal agent with ReAct loop and MCP support. New entrant in the open-source CLI coding-agent space.

### C3 AI
- **C3 Code** — GA released April 8, 2026. Autonomous agentic coding layer on top of C3 Agentic AI Platform; targets business analysts and data scientists building enterprise AI applications from natural-language requirements.

### Emergent
- **Wingman** — Launched April 15, 2026. Messaging-first autonomous AI agent from India-based vibe-coding startup Emergent. Runs background tasks across tools and workflows; positions as an execution layer beyond code generation.

## Open Source

### Gemini CLI (Google)
- First release April 2026. ReAct-loop terminal agent, MCP support, Apache 2.0. Adds Google to the CLI-agent tier alongside Aider, OpenCode, and Claude Code CLI.

### OpenCode
- **v1.3.3** — Updated April 2026. New `opencode export --sanitize` flag to redact PII/confidential transcript data; fixed diff line-number contrast; improved plugin auth. 95K+ GitHub stars — the most-starred dedicated coding agent as of April 2026.

### Aider
- Ongoing maintenance; 39K+ stars, 4.1M+ installs. No major April release but continues as the most battle-tested Git-native CLI option.

### Continue
- Last major update April 10, 2026. 26K+ stars; only open-source tool with full VS Code + JetBrains support.

The densest model release window of 2026 so far: Meta shipped two open-weight multimodal MoE models and its first proprietary frontier model in the same week, Google followed Gemma 4 with an open-source CLI agent, and Anysphere launched its first in-house model alongside Cursor 3. The product layer is converging on parallel agentic sessions (Claude Code desktop, Cursor 3, Windsurf Wave 13), while the open-source CLI tier gains a new major entrant with Gemini CLI.
