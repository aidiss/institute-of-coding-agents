# Releases — 2026-03-09

## Models

### Google — Gemini

- **Gemini 3 Pro Preview** — deprecated today (March 9). Google is retiring the preview tier in favour of the GA **Gemini 3.1 Pro** (released February 19). Gemini 3.1 Pro features a 1M-token context window, 77.1% on ARC-AGI-2 (more than double its predecessor's score), and 94.3% on GPQA Diamond. Full multimodal: text, images, audio, video, and code in a single API surface.

### xAI — Grok

- **Grok 4.20** — shipped recently (late February) with a **parallel agents architecture**, allowing the model to spawn and coordinate multiple agent threads simultaneously. This is the primary architectural differentiator xAI is pushing with this version. Also includes Grok video generation via the Grok Imagine API at 720p with audio (10-second clips).

### DeepSeek

- **DeepSeek V4** — still no official release as of today. Multiple credible sources (Financial Times, TechNode, Reuters) reported a first-week-of-March target timed to China's Two Sessions parliamentary meetings (started March 4). That window has now passed with no announcement on DeepSeek's channels. Expected specs remain: ~1T-parameter MoE with ~32B active parameters, 1M-token context, native multimodal (text, image, video), Apache 2.0 open-weight release. Leaked (unverified) benchmarks: 90% HumanEval, 80%+ SWE-bench Verified.

### Anthropic — Claude

No new releases. Claude Opus 4.6 (Feb 5) and Claude Sonnet 4.6 (Feb 17) remain current. Anthropic holds the top Arena Elo position: Sonnet 4.6 at 1,633 on GDPVal-AA.

### OpenAI — GPT

No new model releases today. GPT-5.4 Thinking (released this week, GA for Plus/Team/Pro) and GPT-5.3 Instant remain current. GPT-5.2 Thinking enters legacy status and will be retired June 5, 2026.

## Products

### Cursor

- **Automations** (shipped March 5, GA) — event- and schedule-triggered agents inside Cursor, driven by Slack, GitHub, Linear, PagerDuty, and webhooks. Bugbot Autofix is the flagship feature: automatically reviews new code additions for bugs and security issues; 35%+ of proposed fixes are now merged. Cursor reported **$2B+ ARR** in February 2026.

### GitHub Copilot

No new feature release today. Since January 2026, Copilot supports **Claude and Codex agents** directly in VS Code and JetBrains alongside its own models — the multi-model pivot is now fully rolled out. Pricing: $10/mo individual, $19/mo business.

### Windsurf

No new release. Wave 13 (December 24, 2025) remains current: parallel multi-agent sessions via Git worktrees, SWE-1.5 free for 3 months, Arena Mode.

### Claude Code (Anthropic)

No new release since the minor stability/Windows patch on March 7.

## Open Source

### Aider

No new release since February 12, 2026. Current version stable on PyPI. Active pairings: Claude Sonnet 4.6, DeepSeek R1/V3, OpenAI o3-mini. Still the only tool treating Git as a first-class citizen with automatic staging and commit message generation.

### OpenHands

No new release since v1.0.0. Cloud and self-hosted deployments active; free tier available via Minimax model with GitHub/GitLab sign-in. SWE-bench Verified resolution rate: 53%.

### SWE-agent

No release activity. Remains research-focused (Princeton/Stanford). Best use case: issue-to-PR automation.

### Devstral (Mistral AI × All Hands AI)

Previously noted: Mistral's **Devstral** — an Apache 2.0 agentic LLM built in collaboration with All Hands AI — remains the strongest open-source model on SWE-bench Verified by a significant margin. Runs on a single RTX 4090 or Mac with 32GB RAM. Scaffolded over OpenHands or SWE-agent.
