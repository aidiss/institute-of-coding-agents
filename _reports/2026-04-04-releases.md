# Releases — 2026-04-04

What shipped this week across models, products, and open source.

---

## Models

**Google Gemma 4** — Released April 2, 2026 (GA, Apache 2.0)
Four open model sizes: E2B, E4B, 26B-A4B (MoE), and 31B. Built on Gemini 3 architecture. Up to 256K token context windows; native vision and audio; 140+ languages. Aimed at agentic workflows and local code generation. Available on Google Cloud and Android AICore Developer Preview. Free under Apache 2.0.

**Alibaba Qwen 3.6-Plus** — Released April 2, 2026 (GA)
1 million token context window by default. Benchmarks against Claude Opus 4.5 on SWE-bench and Terminal-Bench 2.0. Native multimodal: generates front-end code from screenshots and design drafts. Available on Alibaba's Bailian platform at ~$0.29/M input tokens, ~$1.74/M output. Compatible with Claude Code, Cline, and other third-party coding clients.

## Products

**Cursor 3** — Released April 2, 2026 (GA)
Major redesign centered on multi-agent management:
- **Agents Window**: run agents in parallel across local machines, git worktrees, cloud, and remote SSH
- **Design Mode**: annotate and target UI elements in-browser for frontend iteration
- **Agent Tabs**: view multiple chats side-by-side or in a grid
- `/worktree` command creates isolated git worktrees per agent session
- Cloud agents produce mid-task screenshots; sessions hand off from cloud to local instantly
- Underlying "Cursor Composer 2" model built on Kimi K2.5 with custom RL — scores 61.3 on CursorBench (+37% over Composer 1.5)

**GitHub Copilot** — Multiple updates (late March–early April 2026)
- **Organization runner controls** for Copilot cloud agent (April 3): org admins can set and lock a default runner across all repositories
- **Agent mode** now GA in both VS Code and JetBrains
- **50% faster agent startup** via semantic code search (init time cut from ~40s to ~20s)
- **Agentic code review** (March 5): Copilot gathers full project context, passes suggestions to the coding agent to auto-generate fix PRs
- **.agent.md files**: teams can define specialized Copilot agents as markdown files in their repo, visible in the agent picker
- GitHub Copilot in Visual Studio — March update changelog published April 2, 2026

**Claude Code** — Rolling updates (March–April 2026)
- **Computer use** (March 23, Pro/Max): Claude can open files, run dev tools, click, and navigate the screen
- **1 million token context window** now available
- **PowerShell tool** preview for Windows
- **Named subagents** appear in @ mention typeahead
- **"defer" permission** for PreToolUse hooks in headless sessions
- **PermissionDenied hook** fires after auto-mode classifier denials
- `--bare` flag for scripted calls; `--channels` for forwarding approval prompts

**Google Antigravity** — Public Preview (launched November 2025, active development)
Agent-first IDE (VS Code fork). Primary models: Gemini 3.1 Pro and Gemini 3 Flash; also supports Claude Sonnet 4.6, Claude Opus 4.6, and GPT-OSS-120B. Manager Surface for orchestrating multiple async agents. Scores 76.2% on SWE-bench Verified. Free for individuals; paid tier expected later in 2026.

## Open Source

**Claw Code** — Launched ~April 1–2, 2026 (MIT/Apache)
Open-source clean-room reimplementation of the Claude Code agent architecture, written initially in Python and ported to Rust. Created by Sigrid Jin (`instructkr` on GitHub). Crossed 100,000 GitHub stars within days — reported as the fastest-growing repository in GitHub history. A parity mirror is maintained at `ultraworkers/claw-code-parity`. Not affiliated with Anthropic.

**OpenHands V1 / Software Agent SDK** — April 2026 (open source)
OpenHands V0 formally deprecated. V1 built on a new `software-agent-sdk` (Python and REST APIs). Key changes: stateless components with immutable configuration, event-sourced state with deterministic replay, optional sandboxing, MCP integration, and model-agnostic multi-LLM routing. Presented as a poster at MLSys 2026. Cloud-hosted at openhands.dev. Main repo at ~69,500 GitHub stars.

**LangChain open-swe** — New entrant (open source)
`langchain-ai/open-swe`: an open-source asynchronous coding agent from LangChain. No specific release date confirmed this week; appeared as a new entrant in the space.
