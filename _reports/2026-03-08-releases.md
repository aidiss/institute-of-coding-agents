# Releases — 2026-03-08

## Models

### OpenAI — GPT

- **GPT-5.4 Thinking** — now available to ChatGPT Plus, Team, and Pro users, replacing GPT-5.2 Thinking (which remains available until June 5, 2026 under Legacy Models). This is OpenAI's first mainline reasoning model that incorporates the frontier coding capabilities of GPT-5.3-Codex, rolling out across ChatGPT, the API, and Codex. The version bump to 5.4 reflects the addition of Codex-grade coding intelligence. API identifier: `gpt-5.4-thinking`.

- **GPT-5.3 Instant** — now GA for all ChatGPT users and API developers (`gpt-5.3-chat-latest`). Reduces hallucination rates by 26.8% on web-assisted tasks and 19.7% on internal-knowledge-only tasks vs. prior models. Addresses ChatGPT's tendency to preface answers with lengthy caveats before responding.

### DeepSeek

- **DeepSeek V4** — still unconfirmed as of March 8. Multiple sources (Financial Times, TechNode, Reuters) reported a first-week-of-March target; as of today no official release announcement has been made on DeepSeek's channels. Features expected on launch: ~1T-parameter MoE with ~32B active, 1M-token context, native multimodal (text, image, video), Apache 2.0 open-weight release. Leaked unverified benchmarks: 90% HumanEval, 80%+ SWE-bench Verified.

### Anthropic — Claude

No new model releases today. Claude Opus 4.6 (Feb 5) and Claude Sonnet 4.6 (Feb 17) remain current.

- **Security Vulnerability Scanner** (new tool, not a model) — Anthropic launched a standalone tool this week that scans open-source codebases for security vulnerabilities. Reported to have found 500+ bugs that had evaded human security review. Caused notable market reaction (CrowdStrike shares fell ~8% on news).

## Products

### OpenAI — Codex

- **GPT-5.4** is now also available through Codex as a cloud coding agent. Codex runs in isolated VMs, produces merge-ready PRs with video/screenshot artifacts, and is accessible via web, desktop, mobile, Slack, and GitHub. This extends the same capability jump from Thinking to the agentic coding surface.

### Cursor

- **Automations** (shipped March 5, 2026) — Cursor now supports always-on agents triggered by schedules or events from Slack, Linear, GitHub, PagerDuty, and webhooks. Pairs with Cloud Agents (Feb 24) which run in isolated VMs and produce merge-ready PRs with artifacts. Bugbot Autofix (also recent) auto-fixes issues in PRs — 35%+ of its proposed fixes are now merged. Cursor passed **$2B ARR** in February 2026.

### Microsoft Copilot

- **Copilot Tasks** — launched this week as Microsoft's answer to the autonomous-agent wave. Runs scheduled and event-triggered tasks within the Copilot ecosystem. Shipped one day after Perplexity Computer and two days after Notion Custom Agents, reflecting the industry-wide convergence on autonomous agent product surfaces.

### Notion

- **Notion Custom Agents** — GA. Autonomous AI teammates that run 24/7 on schedules and triggers (no per-prompt invocation required). Connect to external tools; designed for recurring knowledge-work workflows (syncing data, drafting summaries, triaging issues). Built on top of Notion's existing database and workflow layer.

### Claude Code (Anthropic)

- Bug-fix release on March 7 addressed startup notification noise for unauthenticated org-registered claude.ai connectors, improved Windows stability, and fixed VS Code session management edge cases. No major feature additions today.

### Windsurf

- No new wave or feature release today. Wave 13 (Dec 24, 2025) remains current with parallel multi-agent sessions via Git worktrees, SWE-1.5 free for 3 months, and Arena Mode.

## Open Source

### Aider

- No new release since February 12, 2026. Stable on current PyPI version with OpenRouter metadata caching, `thinking_tokens`/`reasoning_effort` support, and `--disable-playwright` flag. Best pairings: Claude Sonnet 4.6, DeepSeek R1/V3, OpenAI o3-mini.

### OpenHands

- No new release since v1.0.0. Actively maintained; Azure DevOps integration and `software-agent-sdk` base remain current.

### SWE-agent

- No release activity. Remains research-focused (Princeton/Stanford).

### Zhipu AI — GLM-5

- **GLM-5** shipped recently (late February / early March). 744B parameter MoE, 44B active parameters, 200K context window, 77.8% SWE-bench Verified. Trained on Huawei Ascend chips. Released under MIT license — notable as a strong open-weight coding model from a Chinese lab outside of DeepSeek/Alibaba/ByteDance.
