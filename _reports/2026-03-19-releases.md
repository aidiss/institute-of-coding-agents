# Releases — 2026-03-19

## Models

### Mistral Small 4 — Mistral AI (Mar 16, GA)
Mistral's most significant open release of the month: a **119B-parameter MoE** model (128 experts, 4 active per token ≈ 6B active parameters) under Apache 2.0.

- Consolidates four prior Mistral product lines into one: Mistral Small (instruct), Magistral (reasoning), Pixtral (multimodal), and Devstral (agentic coding).
- 256K-token context window; configurable `reasoning_effort` per request (fast / balanced / thorough).
- 40% lower end-to-end latency and 3× higher throughput vs. Mistral Small 3.
- Day-0 support: Hugging Face, NVIDIA NIM, vLLM, llama.cpp.
- API pricing: **$0.15/M input, $0.60/M output** — positions it squarely as a low-cost frontier alternative.

---

### Mistral Leanstral — Mistral AI (Mar 16–17, GA + free API)
First open-source foundation model specialized for **Lean 4 formal verification** of code.

- Outperforms GLM5-744B, Kimi-K2.5, and Qwen3.5-397B on FLTEval (formal theorem proving eval).
- pass@2 of 26.3 — beats Claude Sonnet 4.6 by 2.6 points at roughly 1/15th the cost ($36 vs. $549 per run).
- Weights on Hugging Face; free API endpoint available with no key required.
- Apache 2.0 license. Reported by The Register (Mar 17).

---

### NVIDIA Nemotron 3 Super — NVIDIA (announced GTC, Mar 11; weights live)
Open-weight model targeting long-context agentic coding tasks.

- 120B total parameters, **12B active per forward pass**.
- 60.47% on SWE-Bench Verified (using OpenHands scaffold) vs. GPT-OSS's 41.90%.
- 91.75% on RULER at 1M tokens.
- 2.2× higher throughput than GPT-OSS-120B; 7.5× higher than Qwen3.5-122B.
- Releases under NVIDIA Nemotron Open Model License with full training recipe.

## Open Source

### OpenHands v1.5.0 — All Hands AI (early Mar 2026, MIT)
Major feature release built on the new `software-agent-sdk` v1.0.0.

- **Planning Agent** with Plan/Code mode switching and automatic `PLAN.md` generation.
- Task list panel, skill slash menus, mid-session Git repository switching.
- **Bitbucket Datacenter** and **Azure DevOps** integration added.
- Native tool calling for GLM and Gemini 3 Pro.
- CLI now installable via `pip install openhands-ai` — no Docker required.
- Cloud platform reports 87% of bug tickets resolved same day.
