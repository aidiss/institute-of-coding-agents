# Releases — 2026-03-21

What shipped this week across the AI coding agent landscape.

---

## Models

**OpenAI — GPT-5.4 mini & GPT-5.4 nano** (March 17, 2026)
- Efficient siblings of GPT-5.4 (which shipped March 5), targeting high-volume and latency-sensitive workloads.
- Availability: GA — ChatGPT subscribers and API.
- Pricing: GPT-5.4 full at $2.50/M input, $20.00/M output; mini/nano pricing unconfirmed.

**Google — Gemini Embedding 2** (March 15, 2026)
- Unified multimodal embedding model covering text, images, video, audio, and documents in a single shared vector space.
- Replaces the need for per-modality embedding models.
- Availability: GA via Google AI / Vertex AI.

**DeepSeek — V3.1 (production as of March 2026)**
- Upgrade to the `deepseek-chat` and `deepseek-reasoner` API endpoints; supports standard and thinking modes.
- Open weights, MIT license; 671B parameters (37B active MoE).
- Pricing: $0.20/M input, $0.77/M output.

**xAI — Grok 4.20** (mid-February, active through March 2026)
- Features a four-agent parallel architecture and real-time X/Twitter data integration.
- Availability: GA via xAI API and Grok.com.

**Meta — "Avocado" delayed**
- Originally targeted for March 2026; pushed to May 2026 after underperformance versus Gemini, GPT-5.4, and Claude 4.6 on reasoning, coding, and writing benchmarks. Meta is reportedly evaluating Gemini licensing as a stopgap.

## Products

**Claude Code — v2.1.63–v2.1.76** (rolling updates this week)
- Voice Mode (`/voice`, push-to-talk via spacebar) — first terminal-native voice coding interface from a major provider.
- `/loop` for recurring tasks, `/effort` with "ultrathink" keyword for max compute, `/color` for session color-coding across parallel terminals.
- MCP Elicitation: structured input from MCP servers mid-task.
- 1M token context window now active.
- Default model upgraded to Claude Opus 4.6.
- 4% of all public GitHub commits now authored by Claude Code (doubled in one month); 82K+ GitHub stars.
- Availability: GA — bundled with Claude Pro/Team/Enterprise.

**Cursor — Composer 2 announced + JetBrains support** (March 2026)
- Composer 2: next-generation in-house agent model for long-horizon autonomous coding tasks — Cursor's first proprietary model, reducing reliance on Claude/GPT APIs.
- JetBrains IDE support shipped, removing VS Code lock-in.
- ARR $2B+ (doubled in 3 months); valuation $29.3B; 1M+ daily users.
- Availability: GA — $20/mo Pro, $200/mo Ultra.
- Source: Bloomberg (March 19) confirmed Cursor is building a model to rival Anthropic and OpenAI directly.

**Augment Code — "Intent" launched** (March 2026)
- Desktop workspace for multi-agent orchestration around a "living spec."
- Coordinator/specialist/verifier architecture; each specialist runs in an isolated git worktree.
- Supports BYOA (Bring Your Own Agent) — compatible with Claude Code, Codex, OpenCode, and others.
- Availability: GA.

**Windsurf (Codeium) — parallel agents shipped** (February–March 2026)
- Shipped parallel agent support as part of the multi-agent wave.
- Acquired by Cognition for $250M following Google's $2.4B hiring of its founding team.
- Ranked #1 on LogRocket's AI dev tool power rankings.
- Pricing: ~$15/mo.

**GitHub Copilot — February 2026 update rolling out**
- Model picker (Claude, Codex, others) and expanded autonomous agent mode.
- 4.7M paid subscribers; 90% of Fortune 500 adoption.
- Pricing: Free (2,000 completions/month), Pro ($10/mo), Business ($19/user/mo), Enterprise ($39/user/mo).

**Perplexity — Personal Computer agent** (announced this week)
- Persistent AI agent running on a dedicated local device (e.g., Mac Mini); accesses local files and apps, controllable remotely.
- Built with audit trails and action approvals.
- Availability: Preview/announced.

**Google Gemini in Workspace** (announced this week)
- Upgraded to use multiple specialized models (Gemini 3, DeepMind tools) to generate full documents, spreadsheets, and presentations by pulling from emails, chats, and Drive files.

## Open Source

**OpenHands — v1.0.0** (activity through March 19, 2026)
- Introduces `software-agent-sdk` (Python + REST APIs).
- New Planning Agent with Plan Mode / Code Mode toggle generating a structured `PLAN.md`.
- Support for ephemeral workspaces in Docker/Kubernetes.
- License: MIT (core); enterprise directory has separate terms.

**DeepSeek-Prover-V2** (March 2026)
- Open-source model for formal theorem proving in Lean 4.
- Open weights license.

**Superdesign** (gaining traction this week)
- Open-source AI product designer living inside VS Code, Cursor, Windsurf, and Claude Code.
- Generates UI mockups and components from natural language.
- Initial launch January 2026; notable uptick in adoption this week.

## Key Themes

- **Multi-agent is now table stakes.** Every major coding tool shipped parallel agent support in the February–March window.
- **Context windows converging at 1M tokens.** GPT-5.4 and Claude Code both now offer 1M-token contexts.
- **Coding tool consolidation.** Windsurf acquired by Cognition; Cursor building its own model. The independent tool layer is compressing.
- **Voice interfaces arriving in dev tools.** Claude Code's push-to-talk voice mode is the first terminal-native voice coding interface.
- **Meta falling behind.** Avocado delay and potential Gemini licensing signals Meta is no longer at the frontier.
