# Releases — 2026-03-14

## Models

### Claude Sonnet 4.6 — Anthropic (Feb 17, GA)
Anthropic shipped **Claude Sonnet 4.6** on February 17, delivering near-Opus performance at Sonnet pricing. On GDPval-AA Elo (expert-level office work), Sonnet 4.6 leads the field at **1,633 points**, ahead of Opus 4.6 and Gemini 3.1 Pro. 200K context window; 1M-token context available in beta on Sonnet 4.

### Claude Opus 4.6 — Anthropic (Feb 5, GA)
Frontier Anthropic model, shipped February 5. Reported **80.9% on SWE-bench**. Available on API and Claude.ai Max/Team/Enterprise plans.

### Gemini 3.1 Pro — Google DeepMind (Feb 19, GA)
Released February 19. Dominates **13 of 16 major benchmarks**, scoring **77.1% on ARC-AGI-2** (more than double Gemini 3 Pro) and **94.3% on GPQA Diamond**. Pricing: $2/M input tokens, $12/M output tokens.

### GPT-5.3 Codex — OpenAI (Feb 5, GA)
OpenAI's latest GPT-5 variant launched February 5 alongside Claude Opus 4.6. Offers **six reasoning levels** (none → xhigh), trading latency for depth.

### Llama 4 — Meta (GA, open-weight)
Meta's Llama 4 family (Scout, Maverick variants) is available. Maverick reported to outperform GPT-4o and Gemini 2.0 Flash on coding, reasoning, and multilingual tasks. Open-weight for self-hosting.

### DeepSeek V4 — DeepSeek (not yet released as of Mar 14)
Expected but not officially shipped. Reported as a trillion-parameter MoE model with native multimodal capabilities (text, image, video, audio) and a 1M-token context window. An unnamed update with improved coding appeared on DeepSeek's site around March 9; the company has not confirmed the V4 name. Prediction markets give ~88% probability of a March release.

## Products

### Claude Code v2.1.75 — Anthropic (Mar 13, GA)
Released yesterday. Key additions:
- **1M context window for Opus 4.6** enabled by default on Max/Team/Enterprise plans
- `/color` command for session prompt-bar theming
- Session name display via `/rename`
- Last-modified timestamps on memory files
- Hook source display (settings/plugin/skill) in permission prompts
- Bug fixes: voice mode activation on fresh installs, model name display after switching

### Claude Code v2.1.74 — Anthropic (Mar 12, GA)
- **Actionable suggestions in `/context` command** — identifies context-heavy tools and memory bloat with optimization tips
- `autoMemoryDirectory` setting for custom auto-memory paths
- Fixed memory leak in streaming API response buffers

### Claude Code earlier March rollout — Anthropic (Mar 3–7, GA)
- Native **voice mode** (`/voice`) launched March 3 (rolling out from 5% of users)
- `/loop` command and **cron scheduling** for recurring prompts within sessions
- `/simplify` and `/batch` bundled slash commands
- **HTTP hooks** (POST JSON to URL, receive JSON back)
- Project configs and auto memory shared across git worktrees of the same repo
- `voice:pushToTalk` rebindable keybinding
- **Claude Code Analytics API** for organizations (aggregated usage metrics, tool stats, cost data)
- Plugin marketplace and admin controls for Team/Enterprise

### GitHub Copilot Agent Mode + VS Code 1.109 (GA)
**Agent Mode** shipped with MCP support, moving Copilot from completion tool toward a full agent. VS Code 1.109 runs **Claude, Codex, and Copilot agents side-by-side** under one $10/month subscription, with separate context windows per agent. Current pricing tiers: Free (50 req/mo), Pro ($10/mo, 300 req), Pro+ ($39/mo, 1,500 req), Business ($19/user/mo), Enterprise ($39/user/mo).

### Windsurf Wave 13 "Shipmas Edition" — Cognition (Dec 24, 2025, GA)
Shipped in late December but still driving adoption in March. Key features:
- **SWE-1.5 model free** for all users (standard throughput) for 3 months
- **True parallel multi-agent sessions** with Git worktrees — up to 5 concurrent Cascade agents on separate branches
- Side-by-side multi-pane Cascade view
- Dedicated Cascade terminal (beta, macOS) using interactive zsh
- System-level rules for enterprise MDM deployment
- Context window indicator showing real-time token usage

Note: Windsurf was acquired by Cognition AI (Devin) after OpenAI's $3B acquisition bid fell through and Google hired Windsurf's CEO and R&D leads.

## Open Source

### OpenHands — Vulnerability Fixer (Mar 3, GA)
OpenHands (formerly OpenDevin) published *The OpenHands Vulnerability Fixer* on March 3: an automated security remediation agent built on the OpenHands SDK. The SDK is a composable Python library for defining agents that run locally or scale to cloud. Free tier available via Minimax model with GitHub/GitLab sign-in. MIT-licensed.
GitHub: [OpenHands/OpenHands](https://github.com/OpenHands/OpenHands)

### Aider (ongoing, open-source CLI)
As of March 2026: **93 releases**, **13,100+ commits**. Remains the most mature terminal-native AI coding agent — write access to Git repos, auto-commits with sensible messages, works with any editor. Supports GPT-5, Claude, and other frontier models via API key. Trade-offs: no GUI, memory limits on very large monorepos.

### SWE-agent (research, open-source)
Princeton/Stanford research tool optimized for GitHub issue-to-PR automation via an Agent-Computer Interface (ACI). More research-focused than production-ready; not accepting generic prompts. MIT-licensed, actively maintained.
