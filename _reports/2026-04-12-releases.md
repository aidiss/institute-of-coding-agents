# Releases — 2026-04-12

## Models

### Anthropic Claude Mythos Preview (Restricted — April 7, 2026)
Anthropic announced Claude Mythos Preview alongside **Project Glasswing**, a gated program to deploy the model specifically for cybersecurity vulnerability research.

- Not publicly available; access restricted to 12 founding partner organizations (AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Linux Foundation, Microsoft, NVIDIA, Palo Alto Networks, Anthropic).
- Found thousands of previously unknown zero-day vulnerabilities across every major OS and browser, including a 27-year-old flaw in OpenBSD and a 16-year-old bug in FFmpeg's H.264 codec (undetected by every prior fuzzer).
- Autonomously exploited a 17-year-old NFS RCE in FreeBSD (CVE-2026-4747) granting unauthenticated root access without human involvement after the initial prompt.
- Anthropic pledged $100M in model-usage credits to Project Glasswing partners, $2.5M to Alpha-Omega/OpenSSF via the Linux Foundation, and $1.5M to the Apache Software Foundation.
- Tom's Hardware noted that the "thousands" figure is based on 198 manually reviewed vulnerabilities; the broader count extrapolates from that sample.

### Meta Llama 4 Scout & Maverick (Open-weight GA — April 5, 2026)
Meta released two new open-weight multimodal models built on a mixture-of-experts (MoE) architecture and available on Hugging Face and llama.com.

- **Llama 4 Maverick**: 17B active parameters, 128 experts (400B total), 1M token context window, natively multimodal (text, images, video).
- **Llama 4 Scout**: Smaller sibling; also multimodal and MoE-based.
- Instruct-tuned Maverick beats GPT-4o and Gemini 2.0 Flash across broad benchmarks; comparable to DeepSeek V3 on reasoning and coding at less than half the active parameters.
- Both released under Meta's Llama 4 Community License (open-weight download, commercial use permitted with restrictions).
- Separate from Muse Spark (released April 8 under Alexandr Wang), which is fully proprietary and serves Meta's consumer apps.

### xAI Grok 4.1 Fast (Enterprise API GA — early April 2026)
xAI added Grok 4.1 Fast to its Enterprise API, offering faster inference for production agentic workloads on top of the Grok 4 series.

- Available to xAI Enterprise API customers.
- Grok 4.20 (SuperGrok and Premium+ subscribers) remains the highest-capability option with a 2M token context window.

## Products

### GitHub Copilot — Custom Agents & Semantic Index (Week of April 6–12, 2026)
A batch of Copilot updates shipped across the week that go beyond the CLI changes covered previously:

- **Custom agents via `.agent.md`**: Developers can define specialized Copilot agents as repository-scoped `.agent.md` files. Each agent gets workspace awareness, tool access, model selection, and MCP connections.
- **Unified semantic codebase index (VS Code)**: The `#codebase` tool is now purely semantic — the previous fuzzy-text-search fallback was removed. VS Code manages a single index automatically with no manual configuration.
- **Org-wide custom instructions GA**: Available to Copilot Business and Enterprise admins; sets default instructions shaping Copilot behavior across all repositories.
- **BYOK in Copilot CLI**: Bring-your-own-key support is now available in the CLI.
- **MCP server allowlist governance**: MCP server usage in Visual Studio now respects GitHub-managed org allowlist policies, giving admins control over which MCP servers are permitted.

Pricing unchanged: Copilot Free, Pro ($10/month, 300 premium requests), Business ($19/user/month), Enterprise ($39/user/month).

### Devin — Enterprise Feature Update (April 3, 2026)
Cognition shipped a bundle of enterprise-targeted features alongside a new PR review tool:

- **Enterprise-Scoped Secrets**: Secrets can now be scoped to the org level rather than per-user.
- **Enterprise ACU Visibility Control**: Admins can control which teams see compute-unit consumption.
- **Enterprise MCP Registry Enforcement**: Admins can restrict which MCP servers Devin sessions may connect to.
- **Enterprise Build Pinning**: Pin specific build configurations across the org to prevent environment drift.
- **Devin Review**: New tool for codebase/PR understanding — analyzes code changes, groups logically connected hunks, and explains each section to augment human code review. Available to all Devin plan tiers.
