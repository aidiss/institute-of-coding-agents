# Releases — 2026-03-11

## Models

### DeepSeek — V4

- **DeepSeek V4** — confirmed launched around **March 3, 2026**. A 1-trillion-parameter Mixture-of-Experts model with only ~32B active parameters per token, maintaining V3's Multi-head Latent Attention (MLA) while adding three new innovations: Manifold-Constrained Hyper-Connections (training stability at scale), Engram Conditional Memory (efficient 1M-token retrieval), and a Lightning Indexer for sparse attention. Native multimodal capabilities: text, image, video, and audio. Expected open-weight release under Apache 2.0. Hardware optimized for Huawei Ascend and Cambricon chips, with early access reportedly withheld from NVIDIA and AMD. Leaked benchmarks: ~90% HumanEval, >80% SWE-bench Verified — unverified by third parties. No official pricing published yet.

### OpenAI — GPT-5.4

- **GPT-5.4** — GA **March 5, 2026**. OpenAI's first mainline reasoning model that incorporates the frontier coding capabilities of GPT-5.3-Codex into a single unified system. Key specs: 1M-token context window; native computer-use at 75.0% OSWorld-Verified (above 72.4% human baseline); 33% fewer false claims and 18% fewer errors vs GPT-5.2. Available via API as `gpt-5.4` and `gpt-5.4-pro`. In ChatGPT, **GPT-5.4 Thinking** replaces GPT-5.2 Thinking for Plus, Team, and Pro users; GPT-5.2 Thinking moves to Legacy and retires June 5, 2026. Pricing: $2.50/1M input, $20.00/1M output (via OpenRouter). Rated "High cyber capability" under OpenAI's Preparedness Framework.

### Google — Gemini 3.1 Flash-Lite

- **Gemini 3.1 Flash-Lite** — preview launched **March 3, 2026** via Gemini API, Google AI Studio, and Vertex AI. Fastest and most cost-efficient model in the Gemini 3 family: 2.5× faster time-to-first-token and 45% higher output speed vs Gemini 2.5 Flash. Benchmarks: 86.9% GPQA Diamond, 76.8% MMMU Pro; top score on 6 of 11 benchmarks, outperforming GPT-5 mini and Claude 4.5 Haiku. Supports multimodal input (text, image, speech, video), 1M-token context window, 64K max output. Includes adjustable thinking levels for cost/quality control. Pricing: **$0.25/1M input, $1.50/1M output** — roughly 1/8th the cost of Gemini 3.1 Pro, and significantly cheaper than Claude 4.5 Haiku ($1.00/$5.00).

### Anthropic — Claude

No new model release. Claude Opus 4.6 (Feb 5, 80.8% SWE-bench Verified) and Claude Sonnet 4.6 (Feb 17, 79.6%) remain current. Claude memory expanded to free-tier users in early March.

### xAI — Grok

No new release. Grok 4.20 (late February, four-agent architecture) remains current. Full public API access still pending; expected Q2 2026.

## Products

### Claude Code — Code Review (Anthropic)

- **Claude Code Review** — research preview launched **March 9–10, 2026**. Multi-agent system that automatically reviews pull requests for logic errors, bugs, and security vulnerabilities. Integrates with GitHub; opens a PR and multiple agents analyze it in parallel, ranking findings by severity and filtering false positives. Larger PRs get more agents allocated ("deeper read"). Average review time: ~20 minutes. Anthropic internal results: substantive comment rate on code changes rose from 16% to 54%; <1% of findings marked incorrect by engineers. Available to **Claude for Teams and Claude for Enterprise** customers. Pricing: **$15–$25 per review** depending on PR complexity, billed on token usage; org admins can set monthly caps and enable per-repo. Deeper security analysis available via Claude Code Security.

### OpenAI — Codex Security

- **Codex Security** — research preview launched **March 6, 2026**. Application security agent that builds context from repositories and system environments to identify complex vulnerabilities, validate findings, and propose remediations. Scanned 1.2M+ commits in its first 30 days, finding 792 critical and 10,561 high-severity issues. Rolling out to ChatGPT Pro, Enterprise, Business, and Edu via Codex web; **first month at no cost**.

### OpenAI — Promptfoo Acquisition

- OpenAI announced acquisition of **Promptfoo** on **March 9, 2026**. Promptfoo detects and mitigates LLM risks including prompt injection, jailbreaking, data leakage, tool misuse, and policy-violating behavior. Technology will be integrated into OpenAI Frontier after closing.

### Cursor — Automations GA

- **Cursor Automations** — GA **March 5, 2026**. Event- and schedule-triggered background agents driven by Slack, GitHub, Linear, PagerDuty, and webhooks. Bugbot Autofix now auto-merges fixes for 35%+ of flagged issues. Cursor ARR: $2B+ as of February 2026.

### Windsurf (Cognition)

No new release. Wave 13 (December 24, 2025) remains current: parallel multi-agent sessions via Git worktrees, SWE-grep, dedicated Cascade terminal (macOS beta). SWE-1.5 free through end of March 2026. Cognition completed Windsurf acquisition in December 2025 ($250M).

### GitHub Copilot

No new release. Multi-model parallel agents (Claude + Codex + Copilot in VS Code 1.109) fully rolled out. 15M+ active developers.

## Open Source

### OpenHands (All Hands AI)

- **Planning Agent** — shipped **March 4–6, 2026** ([blog post March 6](https://openhands.dev/blog/openhands-product-update---march-2026)). New two-mode workflow: **Plan Mode** generates a structured `PLAN.md` outline from a high-level description; **Code Mode** executes the plan. Modes are seamlessly switchable within a single session.

- **v1.4.0** (February 17, 2026) remains the latest stable release: added MiniMax M2.5 model support; fixed resumed-conversation error status display. v1.3.0 shipped February 2, 2026. Project now at 65K+ GitHub stars.
