# Releases — 2026-03-07

## Models

### Anthropic — Claude

- **Claude Opus 4.6** — released February 5. Available on Anthropic API (Max/Team subscribers). Defaults to medium reasoning effort; "ultrathink" keyword re-enables high effort per turn. Opus 4 and 4.1 removed from Claude Code's first-party API.
- **Claude Sonnet 4.6** — released February 17. Near-Opus performance at Sonnet pricing. Leads GDPval-AA Elo at 1,633 points. 200K context window (1M token beta on Sonnet 4). Multimodal (text + images), supports computer use.

### Google — Gemini

- **Gemini 3.1 Pro** — released February 19. Dominates 13 of 16 major benchmarks; 77.1% on ARC-AGI-2 (more than double Gemini 3 Pro), 94.3% on GPQA Diamond. Priced at $2/M input tokens, $12/M output tokens.

### OpenAI — GPT

- **GPT-5.3 Codex** — released February 5. Coding-focused variant of GPT-5. GPT-5.2 offers six reasoning levels (none, minimal, low, medium, high, xhigh).

### xAI — Grok

- **Grok 4.20** — released February 17. Four-agent parallel processing architecture. Modes: Standard, Spicy (reduced filtering for Premium), Extended Thinking. Follows xAI's unconventional version numbering.

### Meta — Llama

- **Llama 4** (Maverick and Scout variants) — available as open-weights. Maverick is a 400B MoE model. Reported to outperform GPT-4o and Gemini 2.0 Flash on coding, reasoning, and multilingual benchmarks.

### DeepSeek

- **DeepSeek V4** — expected to release in the first week of March 2026 (week of March 3), per Financial Times reporting on February 27. Trillion-parameter MoE (~32B active parameters), native multimodal, 1M-token context window. Optimized for Huawei Ascend chips. Ends the separate V-series/R-series distinction; supports both reasoning and non-reasoning tasks. Open-weight release under Apache 2.0 expected. Leaked benchmarks (unverified) claim 90% HumanEval, 80%+ SWE-bench Verified. Not yet confirmed released as of report date.

### Other Notable (February 2026)

- **Qwen 3.5** (Alibaba), **ByteDance Seed 2.0 Lite and Pro**, **MiniMax M2.5**, **GLM-5**, **Mercury 2**, **LongCat-Flash-Lite** — all shipped in February per LLM Stats tracker.

## Products

### Claude Code (Anthropic)

Recent updates (late February / early March):

- `/loop` command for recurring prompts on a set interval (e.g., `/loop 5m check the deploy`); cron scheduling for session-level recurring prompts.
- Voice STT expanded to 20 languages (added Russian, Polish, Turkish, Dutch, Ukrainian, Greek, Czech, Danish, Swedish, Norwegian).
- VS Code extension: spark icon in activity bar listing all sessions; full markdown plan view with comment support; native MCP server management dialog via `/mcp`.
- Performance: prompt input re-renders reduced ~74%; startup memory reduced ~426KB; Remote Control poll rate cut from 1–2s to once per 10 minutes (300x server load reduction).
- Opus 4.6 now defaults to medium effort for Max and Team subscribers.
- Enterprise plans now self-serve (no Sales call required).
- **Cowork**: Claude Code agentic capabilities brought to the Claude desktop app for knowledge work beyond coding, running in an isolated local VM.
- Enterprise Analytics API: programmatic access to org-level usage/engagement data.

### Windsurf + Devin (Cognition acquisition)

- Cognition AI (Devin) acquired Windsurf in December 2025 for ~$250M. At acquisition: Windsurf had $82M ARR, 350+ enterprise customers, 210 employees; Google secured a separate technology licensing deal.
- As of March 2026, Windsurf ranks #1 in LogRocket AI Dev Tool Power Rankings.
- New **Flows** feature: save and share repeatable agentic workflows (e.g., run tests → fix failures → commit) triggered by a single command.
- Windsurf Pro: $15/month.

### Devin 2.0 (Cognition)

- **Devin 2.0** shipped with Interactive Planning and Devin Wiki (auto-indexes repos, generates architecture docs).
- Pricing slashed from $500/month to $20/month Core + $2.25 per ACU (~15 minutes of active work).
- Completes 83%+ more junior-level tasks per compute unit vs. Devin 1.

### GitHub Copilot

- **Agent Mode** shipped with MCP support — moves Copilot from completion tool to agent.
- VS Code 1.109 runs Claude, Codex, and Copilot agents side by side under one subscription with isolated context windows.
- Copilot Workspace plans features from GitHub Issues before opening the IDE.
- 15 million developers active. $10/month.

### Cursor

- Valuation $29.3B, ARR $2B+. Pricing changes (surprise overages, tighter usage limits) have pushed users to evaluate alternatives. Pro: $20/month.

## Open Source

### OpenHands — v1.0.0

- **Version 1.0.0** released, built on the new `software-agent-sdk`.
- New task tracker interface; **Azure DevOps integration**; native tool calling for `gemini-3-pro-preview`.
- CORS support added to Docker sandbox for remote browser access (`WEB_HOST` env var); host networking mode via `OH_SANDBOX_USE_HOST_NETWORK=true`.
- MIT-licensed. Supports Claude, GPT, or any LLM as backend.

### Aider

- Latest release on PyPI: **February 12, 2026**.
- Shell tab completion for file paths and `--edit-format` options.
- OpenRouter model metadata now fetched directly from OpenRouter with local caching; `thinking_tokens` and `reasoning_effort` parameter support for OpenRouter models.
- Updated Gemini model aliases to point to latest preview versions; Gemini 2.5 Pro marked overeager by default.
- `--disable-playwright` flag added; `aider scrape` uses Playwright if available.
- Best model pairings: Claude Sonnet 4.6, DeepSeek R1/V3, OpenAI o3-mini/GPT-4o.

### SWE-agent

- No major release in this period. Remains research-focused (Princeton/Stanford), targeting GitHub issue resolution. Less production-ready than OpenHands; primarily an ACI (Agent-Computer Interface) research platform.
