# Releases — 2026-04-10

## Models

### Claude Mythos Preview (Restricted — April 7, 2026)
Anthropic released Claude Mythos exclusively to 50+ technology and cybersecurity organizations under **Project Glasswing**, with $100M in usage credits distributed to partners. No public release; Anthropic withheld general availability on safety grounds — the first such action by a leading AI lab in roughly seven years.

- **SWE-bench Verified**: 93.9%
- **GPQA Diamond**: 94.6%
- Reportedly capable of autonomously identifying thousands of zero-day vulnerabilities across major operating systems and browsers.
- Available via Vertex AI to partner organizations only.
- A source-code leak in late March 2026 preceded the formal announcement and prompted both the restricted release and the open-source Claw Code reimplementation (see Open Source below).

### Zhipu AI GLM-5.1 (GA — April 7, 2026)
Z.ai released GLM-5.1 under an **MIT license**, with weights on Hugging Face (`zai-org/GLM-5.1` and `GLM-5.1-FP8`). A commercial API is also available via GLM Coding Plan tiers (Max, Pro, Lite).

- **Architecture**: 744B parameter Mixture-of-Experts, 40B active parameters.
- **Context**: 200K tokens; 131K max output.
- **SWE-Bench Pro**: 58.4 — highest reported score, ahead of GPT-5.4 (57.7), Claude Opus 4.6 (57.3), and Gemini 3.1 Pro (54.2).
- Designed for long-horizon agentic coding: can autonomously plan, execute, test, and optimize a single task for up to 8 hours.
- Compatible with Claude Code, Cline, and any OpenAI-compatible toolchain via API.

### Meta Llama 5 (GA — April 8, 2026)
Meta released Llama 5 with open weights, continuing the Llama open-source line distinct from the proprietary Muse Spark model released the same day.

- Native System 2 (deliberate multi-step reasoning) capabilities.
- Up to 5M token context window.
- Weights available on Hugging Face and llama.com.

### Meta Muse Spark (Limited rollout — April 8, 2026)
The first model from **Meta Superintelligence Labs**, led by Alexandr Wang (formerly Scale AI), marks a departure from Meta's open-source stance — Muse Spark is proprietary.

- Optimized for small size and low latency while retaining reasoning capability in science, math, and health domains.
- Native multimodal: perceives what users see, not just text input.
- **GPQA**: 89.5%
- Rolling out inside Facebook, Instagram, WhatsApp, Messenger, and Ray-Ban Meta AI glasses; no standalone API pricing announced.

### GPT-5.4 → Codex (April 9, 2026)
OpenAI updated the Codex agent to run on GPT-5.4, converging the main reasoning and coding-specialist lineages into a single model for the first time. No new model release; a product-level update.

## Products

### Claude Code v2.1.94 – v2.1.98 (April 8–9, 2026)
Four point releases shipped in two days, with significant platform-expansion and security improvements.

- **v2.1.94** (Apr 8): Full Amazon Bedrock support (powered by Mantle); interactive Bedrock setup wizard; default effort level raised from medium to **high** for API-key and enterprise users; per-model cost and cache-hit breakdown in `/cost`.
- **v2.1.96** (Apr 8): Hotfix for Bedrock 403 auth regression introduced in v2.1.94.
- **v2.1.97** (Apr 9): Focus view toggle (`Ctrl+O`) in NO_FLICKER mode; Cedar policy file syntax highlighting; hardened Bash tool permission checks.
- **v2.1.98** (Apr 9): Google **Vertex AI** interactive authentication wizard; subprocess sandboxing with PID namespace isolation; W3C `TRACEPARENT` environment variable support for distributed tracing.

### Cursor Bugbot — Self-Improvement Update (April 9, 2026)
Cursor's automated code-review agent reached new milestones since its February 2026 launch.

- **Resolution rate**: ~80% (up from 52% at launch).
- **Adoption**: 110,000+ repos enabled; 44,000+ learned rules generated from repository history.
- MCP server support added to code reviews.
- No pricing change; included in Cursor Pro ($20/month) and Pro+ ($60/month).

### GitHub Copilot CLI v1.0.22 (April 9, 2026)
Final release in the v1.0.13–v1.0.22 run that shipped across April 1–9. Key changes this week:

- **v1.0.18** (Apr 4): Critic agent — automatically reviews plans and complex implementations with a complementary model before execution.
- **v1.0.19** (Apr 6): MCP enable/disable settings now persist across sessions.
- **v1.0.20–21** (Apr 7): OpenTelemetry monitoring; new `copilot mcp` command for managing MCP server lifecycle.
- **v1.0.22** (Apr 9): MCP tools with non-standard JSON schemas sanitized for compatibility across all model providers.

### Windsurf Adaptive Model Router (GA — April 6, 2026)
Released in Windsurf Editor v1.9600.38; hotfix v1.9600.41 followed on April 7.

- Available to Pro, Max, and Teams self-serve subscribers.
- The **Adaptive** routing option intelligently selects the optimal model per task, extending quota by directing simpler tasks away from premium models.
- Enhanced model picker shows per-model token pricing, prompt cache timer, and token counts in response cards.
- Overage pricing: $0.50/M input tokens, $2.00/M output tokens, $0.10/M cache reads (over a 2-week rolling window).

## Open Source

### Claw Code (New entrant — April 2, 2026)
A clean-room reimplementation of the Claude Code agent harness architecture, created by Sigrid Jin following the March 2026 Claude Mythos/Claude Code source-code leak. Written in Python (Rust port in progress); not affiliated with Anthropic.

- Reimplements the tool system, query engine, multi-agent orchestration, and memory management without copying proprietary code.
- **72,000+ GitHub stars and 72,600+ forks** within days of launch — one of the fastest-growing open-source AI tool repositories on record.
- Available on GitHub; MIT license.

### OpenHands v1.6.0 (GA — March 30, 2026)
Released just before the reporting window; remains the current stable release as of April 10.

- Hooks support; Catalan language addition; `/clear` command for v1 conversations; copy buttons in code blocks; global skills enable/disable; `/new` slash command menu.
- Bug fixes: conversation persistence, "Export Conversation" duplication, CVE patches via dependency updates.
- ~10 new contributors.

### Gemini CLI (Weekly stable — April 8, 2026)
Google's open-source terminal agent (Apache 2.0) continues weekly releases. The April 8 stable build adds incremental MCP and tool-call reliability improvements on top of the v0.36.0 foundation.
