# Releases — 2026-04-08

## Models

### Meta Llama 4 Scout & Maverick (GA — April 5, 2026)
Meta released two new open-weight multimodal models built on a mixture-of-experts (MoE) architecture. Both process text, images, and video natively.

- **Llama 4 Maverick**: 17B active parameters across 128 experts (400B total). Benchmarks competitive with GPT-4o and Gemini 2.0 Flash. Available on Hugging Face and llama.com.
- **Llama 4 Scout**: Smaller, more efficient sibling model. Also open-weight.
- Availability: Free download (open weights). API access via major providers (Oracle OCI, Google Vertex AI, OpenRouter).

### GPT-5.4 (Released March 5, 2026 — now broadly available)
OpenAI's latest iteration. Record scores on OSWorld-Verified, WebArena Verified, and 83% on GDPval knowledge work benchmark. Computer-use capabilities significantly improved.

### Gemini 3.1 Pro & Flash-Lite
Google released Gemini 3.1 Pro (February/March), topping reasoning benchmarks with 94.3% on GPQA Diamond. Gemini 3.1 Pro (Low and High) now available in Windsurf as a preview for paid subscribers.

### Claude Sonnet 4.6 & Opus 4.6 (Released February 2026)
Anthropic's current generation. Sonnet 4.6 leads the GDPval-AA Elo benchmark at 1,633 points — near-Opus performance at Sonnet pricing.

## Products

### Cursor 3 (GA — April 2, 2026)
Major version release repositioning Cursor as an agent-first IDE.

- **Agents Window**: Run multiple agents in parallel across repos and environments (local, git worktrees, cloud, remote SSH).
- **Design Mode**: Annotate and target UI elements directly in the browser for precise visual feedback.
- New commands: `/worktree` (isolated git worktree per task), `/best-of-n` (run same task across multiple models in parallel, compare outcomes).
- Pricing: $20/month Pro tier unchanged; all new features included.

### GitHub Copilot CLI v1.0.20 (April 7, 2026)
- Reduced UI sluggishness during live response streaming.
- Spinner stays active until background agents and shell commands finish.

### GitHub Copilot CLI v1.0.19 (April 6, 2026)
- `/mcp enable` and `/mcp disable` commands now persist across sessions.
- OpenTelemetry monitoring improvements.

### GitHub Copilot CLI v1.0.18 (April 4, 2026)
- New **Critic agent**: automatically reviews plans and complex implementations using a complementary model to catch errors before execution.

### GitHub Copilot SDK (Public Preview — April 2, 2026)
Available to all Copilot and non-Copilot subscribers (including Copilot Free and BYOK enterprises).

- Fine-grained system prompt customization (replace/append/prepend/transform callbacks).
- Token-by-token streaming.
- Blob attachments (images, screenshots, binary data inline).
- Built-in OpenTelemetry distributed tracing.

### GitHub Copilot — Additional April Updates
- **@copilot in PRs**: Copilot can fix failing workflows, address review comments, and push updates back to the PR in its own cloud environment.
- **Copilot Coding Agent Management APIs** (Public Preview): Org owners can programmatically control repository access at scale.
- **Dependabot + AI agents**: Dependabot alerts can now be assigned to Copilot, Claude, or Codex for automated remediation.

### Windsurf — April Updates
- `.codeiumignore` and `.gitignore` support for Fast Context.
- Enterprise: enforce ignore rules globally via `~/.codeium/.codeiumignore`.
- Gemini 3 Pro (Low/High) added as preview for paid Trial, Pro, and Teams subscribers.
- UI alignment fixes and general stability improvements.

## Open Source

### Aider — Recent Releases (early April 2026)
- Added support for `gemini-2.5-pro`, `gemini-2.5-flash`, and `gemini-2.5-pro-preview-06-05` with thinking tokens.
- Updated aliases: `flash` → `gemini-2.5-flash`, `gemini` → `gemini-2.5-pro`.
- Responses API support for `o1-pro` and `o3-pro` (OpenAI).
- `--add-gitignore-files` flag: allows editing files normally excluded by `.gitignore`.
- `--commit-language` option: specify language for auto-generated commit messages.
- Thinking tokens can now be disabled by setting to `0`.
- Deepseek models: max tokens raised to 65,536.
- Aider self-reports writing 21% of its own code in this release cycle.

### OpenHands v1.6.0 (Released March 30, 2026)
- Kubernetes support added.
- **Planning Mode** (beta): new Planning Agent generates a structured `PLAN.md` before switching into Code Mode.
- Seamless toggle between Plan Mode and Code Mode.
- Built on the new `software-agent-sdk` with broad performance optimizations.
