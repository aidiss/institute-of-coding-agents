### Grok 4.20 — xAI (Feb 17, GA — highlighted this week)
xAI's current flagship, released February 17, continues to be a notable talking point in the March 17–18 coverage cycle as a top-five frontier model. Key differentiator is a **four-agent parallel processing architecture**:

- **Grok** — overall response coordination

### Qwen 3.5 Small Series — Alibaba Cloud (Mar 2, GA)
Alibaba's Qwen team completed a 16-day rollout with the **Qwen 3.5 Small** series (0.8B–9B parameters). Released under Apache 2.0 on Hugging Face and ModelScope.

- Completes the full Qwen 3.5 family: flagship 397B-A17B MoE (Feb 16), Medium series (Feb 24–26), and now Small.

Apache 2.0 licensed across the full Small series.

## Products

### Claude Memory — Anthropic (Mar 2, now GA for all tiers)
Anthropic completed the memory paywall removal, making cross-conversation memory free for all Claude users including the free tier. The March 17 news cycle highlighted this as the week's most significant Anthropic development.

- **Automatic memory generation:** Claude infers and stores your name, writing style, preferences, and ongoing project context in a user-editable text file.
- **Memory import tool:** Allows importing interaction history from rival platforms (ChatGPT, Gemini) — a data portability "digital passport" for AI users switching products.
- **Pause controls:** Users can pause memory at any time without losing stored context.
- Free tier users up **60% YTD**; paid Pro/Max subscribers have **doubled**.
- Eight-month staged rollout completed; previously a $20/month Pro feature.

Directly competes with ChatGPT's paid-tier memory, which has existed for over a year.
