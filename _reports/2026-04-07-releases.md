# Releases — 2026-04-07

What shipped this week across models, products, and open source.

---

## Models

**Mistral Small 4** — Released early April 2026 (GA)
First Mistral model to unify three previously separate model lines — Magistral (reasoning), Pixtral (multimodal vision), and Devstral (agentic coding) — into a single dense model. Targets the sub-30B class for local deployment and API use. Available via Mistral API and La Plateforme.

**Mistral Voxtral TTS** — Released April 2026 (GA)
Mistral's first text-to-speech model. Supports 9 languages with low-latency streaming and custom voice cloning. Pricing: $0.016 per 1,000 characters via API. Available on La Plateforme.

**Mistral Leanstral** — Released early April 2026 (GA, Apache 2.0)
Open-source 6B-parameter agent purpose-built for Lean 4 formal verification. Zero-setup integration with Mistral Vibe; free tier via Mistral Labs API; MCP server support. First dedicated formal-methods agent from a major lab.

**Alibaba Qwen 3.6-Plus** — Released April 2, 2026 (GA, proprietary)
1M token context window, always-on chain-of-thought reasoning, native function calling. Optimized for agentic coding and long-document reasoning. Available via Alibaba Cloud Model Studio and free preview on OpenRouter. Claimed parity with Claude Opus 4.5 on SWE-bench and GPQA score of 0.904.

**Google Gemma 4** — Released April 2, 2026 (GA, Apache 2.0)
Four open-weight sizes: E2B, E4B (edge/mobile), 26B MoE, 31B Dense. 256K context, multimodal (vision + audio), 140+ languages. E2B and E4B run fully offline on Raspberry Pi and NVIDIA Jetson Orin Nano. Weights on Hugging Face, Kaggle, and Ollama; 31B and 26B MoE in Google AI Studio.

**Zhipu AI GLM-5V-Turbo** — Released April 2, 2026
High-throughput multimodal vision model from Chinese lab Zhipu AI. API access with a Fast inference tier. Limited benchmark data published at launch.

## Products

**Claude Code v2.1.89–v2.1.92** — Released April 1–4, 2026 (GA)
Four point releases in four days:
- **v2.1.89** (Apr 1): Faster session resume, new hook events (defer permission decisions, PermissionDenied hooks), flicker-free rendering (`CLAUDE_CODE_NO_FLICKER=1`), MCP connection optimization.
- **v2.1.90** (Apr 2): Interactive `/powerup` lessons, offline marketplace caching, expanded protected directories. Fixed infinite rate-limit dialog loop and prompt-cache misses after resume.
- **v2.1.91** (Apr 3): MCP result persistence up to 500K characters, multi-line prompt support in deep links, improved edit efficiency with shorter anchors.
- **v2.1.92** (Apr 4): Interactive AWS Bedrock setup wizard (auth, region, credential verification, model pinning), per-model cost breakdowns in session summary, 60% faster large-file writes, `/release-notes` interactive version picker.

**Cursor 3** — Released April 2, 2026 (GA)
Major rebuild around multi-agent orchestration: Agents Window for parallel local/cloud/SSH/worktree agents, Design Mode for in-browser UI annotation, multi-LLM comparison shortcut, and `/worktree` command for isolated parallel agent branches. Pricing unchanged: Pro $20/mo, Pro+ $60/mo, Ultra $200/mo.

**Windsurf v1.9600.1032** — Released April 2, 2026 (GA)
Token usage tracking added to the response card — per-response breakdown of input, output, and cached tokens with a context window indicator showing prompt-cache expiry. Recent broader updates include Fast Context (SWE-grep, up to 20× faster code retrieval at 2,800+ tokens/sec) and GPT-5.4 support with adjustable reasoning effort.

**GitHub Copilot** — Multiple updates, April 2026
- Copilot CLI removed gpt-5.1-codex model support (Apr 1); added new MCP server configuration commands.
- Copilot for Jira expanded: model selection from within Jira, ticket references in PRs, Confluence context via MCP.
- Enterprise/org admins now have Copilot coding agent activity in daily and 28-day usage reports.
- **Policy change** (effective April 24, 2026): interaction data from Free/Pro/Pro+ users will be used for model training by default; opt-out available in settings. Business and Enterprise tiers unaffected.
- Coding agent is now powered by Claude Sonnet 4.6.

**Google AI Edge Gallery (Gemma 4)** — Released April 6, 2026 (GA, free)
iOS and Android app updated to run Gemma 4 E2B and E4B fully on-device. New features: Thinking Mode (step-by-step reasoning display) and Agent Skills (Wikipedia fact-grounding, maps integration, visual summaries).

**ChatGPT CarPlay voice mode** — Released April 2, 2026 (GA)
Hands-free voice chat for supported CarPlay vehicles. Requires iOS 26.4+. Conversations can be started or resumed from the car.

## Open Source

**Mistral Leanstral** — April 2026 (Apache 2.0)
6B-parameter Lean 4 formal verification agent, zero-setup in Mistral Vibe, free Labs API, MCP support. First open-source model in the formal-methods coding category from a major lab. (See Models for full details.)

**Google Gemma 4** — April 2, 2026 (Apache 2.0)
E2B, E4B, 26B MoE, and 31B Dense weights fully open under Apache 2.0. (See Models for full details.)

**OpenHands SDK** — Ongoing (MIT)
OpenHands control-plane architecture is fully open-sourced under MIT license. The team published architecture documentation in early April covering cloud-based agent orchestration enabling thousands of parallel coding agents. Main repo at ~69,500 GitHub stars. V0 deprecated; V1 SDK redesign active.

**DeepSeek-V3.2** — Current (MIT)
Updated deepseek-chat and deepseek-reasoner endpoints running V3.2 weights. MIT licensed, available open-weight on Hugging Face. DeepSeek V4 (~1T parameter MoE, 1M context, multimodal) has been observed on API nodes but has not been publicly released as of April 7.
