# Releases — 2026-03-12

## Models

### Alibaba — Qwen 3.5 Small Series

- **Qwen 3.5 Small Series** — released **March 1, 2026**. Four dense model variants at 0.8B, 2B, 4B, and 9B parameters. The 9B model matches GPT-OSS-120B (a model 13× its size) on standard benchmarks; the 2B model runs locally on a recent iPhone in airplane mode with 4 GB RAM. Open weights under Apache 2.0. Positioned as an edge/on-device tier complementing the larger Qwen 3.5 flagship.

### Zhipu — GLM-5

- **GLM-5** — released approximately **March 1–8, 2026**. 744B-parameter Mixture-of-Experts model with 44B active parameters, 200K context window. Reports **77.8% on SWE-bench Verified**, placing it among the top open-weight coding models. Trained on Huawei Ascend chips; MIT license. Open weights available via Hugging Face.

### DeepSeek — V4 (no official release)

- Still unreleased as of March 12, 2026. On March 9 a website model update was spotted that some users labelled "V4 Lite," but DeepSeek has not confirmed it. Multiple predicted windows (Lunar New Year, late-February, early-March, mid-March) have passed without an official launch. Architecture specs remain circulating: 1T-parameter MoE, ~32B active params, native multimodal (text/image/video/audio), optimized for Huawei Ascend/Cambricon hardware. Unverified leaked benchmarks: ~90% HumanEval, >80% SWE-bench Verified.

## Products

### Claude Code — Voice Mode, Loop, and HTTP Hooks (Anthropic)

- **Voice Mode** — shipped **March 3, 2026** (`/voice` in CLI). Push-to-talk key is rebindable via `keybindings.json`; transcription tuned for developer vocabulary (regex, OAuth, JSON). STT now supports 20 languages (added Russian, Polish, Turkish, Dutch, Ukrainian, Greek, Czech, Danish, Swedish, Norwegian).
- **`/loop` command** — recurring prompt/command scheduling (e.g., `/loop 5m check the deploy`). Paired with cron scheduling tools and a `CLAUDE_CODE_DISABLE_CRON` env var to kill scheduled jobs mid-session.
- **HTTP hooks** — hook config now supports `"type": "http"` to POST JSON to a URL and receive JSON back, instead of running a shell command; custom headers with env var interpolation supported.
- **Slash command additions** — `/simplify`, `/batch`, `/claude-api` bundled. `/model` shows active model in the menu. Effort levels simplified to low/medium/high (removing "max") with new symbols (○ ◐ ●).
- **Bash auto-approvals** — `lsof`, `pgrep`, `tput`, `ss`, `fd`, `fdfind` added to the auto-approval allowlist to reduce permission prompts for read-only operations. Bash parsing switched to native module for faster init and no memory leak.
- **Bug fixes** — fixed API 400 errors with third-party `ANTHROPIC_BASE_URL` gateways; fixed symlink bypass in `acceptEdits` mode; fixed deadlock when many skill files change at once.

### ChatGPT for Excel (OpenAI)

- **ChatGPT for Excel** — in beta as of early March 2026. Excel add-in that builds, updates, and analyzes spreadsheet models using GPT-5.4 Thinking. Includes financial data integrations. Rolling out to select Business plan users in the US, Canada, and Australia.
