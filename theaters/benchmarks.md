# THEATER: BENCHMARKS

## Mission
Track the competitive evaluation landscape. Who leads, by how much,
and is progress accelerating or stalling?

## Search for
- Current SWE-bench Verified leaderboard (top 10 with scores)
- SWE-bench Lite, HumanEval, LiveCodeBench latest results
- Any new benchmarks announced or released
- Methodological disputes or benchmark gaming concerns

## Output format

```
## Benchmarks
_As of YYYY-MM-DD_

### Leaderboard (SWE-bench Verified)
| Rank | Actor | Score | Date reported |
...

### Other Benchmarks
...

### Analyst Note
One paragraph: what the numbers mean, trend direction, any caveats.

### Sources
- [source](url)
```

## Rules
- Always include the date scores were reported — benchmark claims go stale fast.
- Flag if a result is unverified or self-reported.
