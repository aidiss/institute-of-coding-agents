# Releases — 2026-03-31

## Models

**Claude Mythos / "Capybara" (Anthropic) — leaked, not yet shipped**
An unreleased next-generation Anthropic model was exposed via a publicly accessible data store on March 26. Anthropic confirmed the drafts were real, describing it as "a step change" and "the most capable we've built to date," with dramatically higher scores than Claude Opus 4.6 on coding, academic reasoning, and cybersecurity benchmarks. Two name candidates appear across draft versions: Mythos and Capybara. Planned for a slow rollout starting with a small group of cybersecurity-focused early-access customers. No GA date announced.

**GPT-5.4 (OpenAI) — GA March 5**
Three variants: Standard, Thinking (reasoning), and Pro. 1.05M token context window. 57.7% on SWE-Bench Pro, 83% on GDPval knowledge-work benchmark, 33% fewer hallucinated claims vs. GPT-5.2. Available across ChatGPT (Plus/Team/Pro) and API simultaneously; GPT-5.4 mini rolled out to Free/Go users. GPT-5.1 models retired March 11.

**Gemini 3.1 Flash-Lite (Google) — preview, March 3**
Available via Gemini API and Vertex AI. Pricing: $0.25/1M input, $1.50/1M output. 2.5× faster time-to-first-token vs. 2.5 Flash, 45% faster output throughput. Note: `gemini-2.5-flash-lite-preview-09-2025` reached end-of-life March 31.

**Gemini 3.1 Pro (Google) — rolling out**
Rolling out globally in the Gemini app with higher limits for AI Pro/Ultra subscribers. Computer Use tool launched for `gemini-3-pro-preview` and `gemini-3-flash-preview`.

**Mistral Voxtral TTS — GA March 26**
Mistral's first TTS model. 4B parameters, runs on consumer hardware. 9 languages, 90ms time-to-first-audio, custom voice cloning from under 5 seconds of audio. Matches ElevenLabs Flash v2.5 in human evaluations. Available on Hugging Face under Creative Commons.

**Mistral Small 4 — mid-March**
119B total / 6B active MoE. 256k context. 40% lower latency and 3× higher throughput vs. Mistral Small 3. Unifies Magistral (reasoning), Pixtral (multimodal), and Devstral (coding) in one model. Apache 2.0 license.

**Leanstral (Mistral) — March 2026**
First open-source Lean 4 code agent for formal verification. 6B parameters, Apache 2.0, MCP support, zero-setup via Mistral Vibe.

**grok-code-fast-1 (xAI) — March 2026**
xAI's first dedicated coding model: "a speedy and economical reasoning model that excels at agentic coding." First Code Model designed for direct integration with code editors.

**Grok 4.20 Beta 0309 (xAI) — API, March 10**
2M token context. $2.00/$6.00 per 1M tokens. 231.9 tokens/sec output speed.

**NVIDIA Nemotron 3 Super — announced GTC March 11**
120B total / 12B active MoE parameters. Open weights. 60.47% on SWE-Bench Verified (OpenHands scaffold). Positioned as an enterprise coding agent foundation model.

**DeepSeek V4 — not yet released**
Reported 1T-parameter MoE model with 1M context and native multimodal (text/image/video). A stealth context-window expansion to 1M tokens appeared around March 9 with no announcement. An anonymous model "Hunter Alpha" surfaced on OpenRouter March 11 with matching specs — unconfirmed. Official release still pending.

## Products

**Claude Code (Anthropic) — v1.0.12, March 28**
Large reliability/performance release:
- `X-Claude-Session-Id` header for proxy session aggregation
- `.jj` and `.sl` VCS directories excluded from Grep/autocomplete
- Faster `--resume` startup; plugin disk caching; AI-generated Remote Control session titles
- WebFetch now identifies as `Claude-User` for robots.txt allowlisting
- Non-streaming fallback token cap raised 21k → 64k; timeout raised 120s → 300s
- Fixes: `deniedMcpServers` bypass, multi-monitor computer-use crash, memory leak in interrupted remote streaming, critical `PreToolUse` hook bypass circumventing enterprise deny rules

March 29: targeted fix restoring Cowork Dispatch message delivery. March 21: Opus 4.6 default output limits raised to 64k (upper bound 128k).

**Cursor Composer 2 — GA March 19**
Cursor's own proprietary coding model. 200k token context. 60%+ on CursorBench. Pricing: $0.50/1M input, $2.50/1M output. Trained exclusively on coding data.

**Cursor Self-Hosted Cloud Agents — March 25**
Code, builds, and secrets stay entirely within customer infrastructure. Same capabilities as Cursor-hosted agents (isolated VMs, full dev environments, multi-model harnesses).

**Cursor Automations — March 5**
Trigger agents automatically from code pushes, Slack messages, or timers; BugBot reviews every new commit.

**GitHub Copilot CLI v1.0.12 — March 26**
Fixes for `.mcp.json` server startup at git root, Windows clipboard, `/diff` rendering, plugin hook env vars, workspace MCP server loading, and `/clear` preserving MCP servers. Coding agent now starts 50% faster; GPT-5.3-Codex long-term support added; Gemini 3 Pro deprecated in Copilot.

**Devin (Cognition) — March 27 update**
"Preview upcoming features" toggle with streaming thoughts and faster execution; Inline File Previews (HTML/PDF/SVG rendered in sidebar); Focus Mode. Also: Windsurf acquired by Cognition (after Google hired CEO, co-founder, and ~40 senior engineers), bringing Windsurf IDE capabilities, brand, ~210 employees, and $82M ARR under the Devin umbrella.

**Augment Code — active shipping**
Key recent features: MCP OAuth Paste Mode over SSH, Effort Levels for reasoning, Enter Plan Mode Tool mid-conversation, parallel tool execution, JetBrains 2026.2 EAP support. Claims 51.80% on SWE-Bench Pro (top among all entrants); 70.6% on SWE-Bench Verified with Claude Sonnet 4. First ISO/IEC 42001 certified AI coding assistant.

**JetBrains Central — announced March 24**
Control layer for agentic workflows connecting JetBrains Air (agentic IDE), the Junie LLM-agnostic coding agent, and development infrastructure. Early access planned for Q2 2026.
