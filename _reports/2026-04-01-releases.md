# Releases — 2026-04-01

## Models

### Anthropic — Claude

- **Claude Opus 4.6** — GA (released early March 2026). 1M-token context window. SWE-Bench Verified score of 80.8%, highest of any commercial model. Introduced multi-agent parallelism in Claude Code. Pricing follows existing Opus tier rates. Default output raised to 64k tokens, ceiling to 128k.
- **Claude Sonnet 4.6** — GA. Near-Opus performance at Sonnet pricing. Leads GDPval-AA Elo benchmark at 1,633 points; users preferred it over prior Sonnet in 70% of head-to-head comparisons. Ceiling also raised to 128k output tokens.

### OpenAI — GPT

- **GPT-5.4 "Thinking"** — GA (released March 5, 2026). Achieves GPT-6-level reasoning in a smaller architecture. Available on ChatGPT Plus ($20/mo), Team, and Pro tiers. API pricing: $2.50/$15 per million input/output tokens (standard); $30/$180 per million tokens (Pro tier). SWE-Bench: 77.2%.
- **GPT-5.1 Codex-Max** — GA. Frontier-grade agentic coding variant optimized for long-horizon engineering; powers the OpenAI Codex cloud agent.

### Google DeepMind — Gemini

- **Gemini 3.1** — GA (released early March 2026). Three-tier lineup: Flash-Lite (speed/cost), Pro ($2/$12 per million tokens), Deep Think (heavy reasoning and math). ARC-AGI-2 score of 77.1%, more than doubling Gemini 3 Pro. Leads Artificial Analysis Intelligence Index at 57 points (tied with GPT-5.4 Pro). Includes native Google Search integration.

### xAI — Grok

- **Grok 4** — GA. Targeted at research, data analysis, and expert-level problem-solving.
- **Grok 4.1 Fast** — GA. 2M-token context window; leads long-context benchmarks.
- **Grok Code Fast 1** — GA. Speed/cost-optimized variant for software development.
- **Grok 3** — GA. Available via X Premium at $50/month.

### Meta — Llama

- **Llama 4 Scout** and **Llama 4 Maverick** — GA (released earlier in 2026, still current). Open-weight, natively multimodal (text, images, short video). MoE architecture. Scout features a 10M-token context window.

### DeepSeek

- **DeepSeek V4** — GA. 1-trillion-parameter open-weight model. Pricing: $0.55/million input tokens for R1 reasoning variant. Aimed at enterprises with data sovereignty requirements.
- **DeepSeek-Prover-V2** — GA. Open-source model for formal theorem proving in Lean 4.

## Key Themes

- **Post-avalanche integration**: The March 10–16 window saw 12 models launch in one week across 6 companies. Late March has been quieter, with teams absorbing the new releases.
- **Full agency is now the baseline**: Every major coding tool ships autonomous agent modes. Differentiation has shifted to orchestration, parallelism, and oversight UX.
- **Context windows continue expanding**: Llama 4 Scout (10M), Grok 4.1 Fast (2M), Claude Opus 4.6 (1M), Gemini 3.1 (1M+).
- **Open-weight parity on benchmarks**: DeepSeek V4, Llama 4, and Mistral are now competitive with closed frontier models on specific tasks.
- **Consolidation**: Google acquired Windsurf/Codeium (~$2.4B). The coding tool market is narrowing around a few well-capitalized players.
