# Releases — 2026-03-10

## Models

### DeepSeek

- **DeepSeek V4** — no official release as of today. The Financial Times reported a "first week of March" window timed to China's Two Sessions parliamentary meetings (started March 4). That window has now passed with no announcement on DeepSeek's official channels. Prediction markets still give an 88% probability of a March 2026 release. Expected specs: ~1T-parameter MoE with ~32B active parameters, native multimodal (text, image, video), 1M-token context window, Apache 2.0 open-weight release, and Engram memory architecture for long-context retrieval. Hardware reportedly optimized for Huawei and Cambricon chips, not NVIDIA.

### OpenAI — GPT

- **GPT-5.4 Thinking** — GA this week for ChatGPT Plus, Team, and Pro users. Replaces GPT-5.2 Thinking, which moves to "Legacy Models" and retires June 5, 2026. No new release today.
- **ChatGPT for Excel** — beta launched in early March. An Excel add-in that builds, updates, and analyzes models directly in workbooks; uses GPT-5.4 Thinking for finance workflows; rolling out to select US, Canada, and Australia business plans.

### Anthropic — Claude

No new model releases. Claude Opus 4.6 (Feb 5, 80.8% SWE-bench Verified) and Claude Sonnet 4.6 (Feb 17, 79.6%) remain current. Anthropic expanded **Claude memory** to free-tier users in early March — previously a paid-plan feature.

### Google — Gemini

No new release. **Gemini 3.1 Pro** (GA February 19) remains current: 1M-token context window, 77.1% ARC-AGI-2, 94.3% GPQA Diamond. Google kept pricing identical to Gemini 3 Pro.

### xAI — Grok

No new release. **Grok 4.20** (late February) with parallel-agents architecture remains current. Full API access still not open; expected Q2 2026.

## Products

### OpenAI Codex

- **Codex outage — March 9.** Codex (cloud coding agent) experienced degraded service for a portion of users on March 9. Reports peaked around 10:44 AM ET per DownDetector. Service has since recovered. No root-cause post-mortem published yet.

### Cursor

No new release today. **Automations** (shipped March 5, GA) — event- and schedule-triggered background agents driven by Slack, GitHub, Linear, PagerDuty, and webhooks — remain the latest feature. Bugbot Autofix now auto-merges fixes for 35%+ of flagged issues. Cursor ARR: $2B+ as of February 2026.

### GitHub Copilot

No new release. The multi-model pivot (Claude, Codex, and Copilot agents running in parallel inside VS Code 1.109 and JetBrains) is fully rolled out. 15M+ active developers; $10/mo individual, $19/mo business.

### Windsurf (Cognition)

No new release. Wave 13 (December 24, 2025) remains current: parallel multi-agent sessions via Git worktrees, SWE-grep, and SWE-1.5 free for 3 months. Cognition completed the Windsurf acquisition in December 2025 ($250M); integration of Devin capabilities into the Windsurf IDE is ongoing.

### Claude Code (Anthropic)

No new release since the minor stability/Windows patch on March 7. Remains the terminal-based benchmark leader (Opus 4.6, 80.8% SWE-bench Verified). Pricing: $20/mo base, $150–200/mo for heavy Opus usage.

## Open Source

### OpenHands (All Hands AI)

- **Planning Agent** — released March 4–6, 2026 ([blog post March 6](https://openhands.dev/blog/openhands-product-update---march-2026)). New two-mode workflow: **Plan Mode** generates a structured `PLAN.md` from a high-level description; **Code Mode** executes the plan. Switching between modes is seamless within a single session. Also shipped this week: *Vulnerability Fixer* tool (March 3) for automated security remediation, and *Learning to Verify AI-Generated Code* guidance post (March 5). OpenHands v1.0 remains on SWE-bench Verified at 53% resolution rate; free tier available via Minimax model.

### Aider

No new release since February 12, 2026. Stable on PyPI. Active pairings: Claude Sonnet 4.6, DeepSeek R1/V3, OpenAI o3-mini. Git-native workflow (automatic staging, commit message generation) remains its primary differentiator.

### SWE-agent

No release activity. Research-focused (Princeton/Stanford). Best use case: issue-to-PR automation.

### Devstral (Mistral AI × All Hands AI)

No new release. Apache 2.0 agentic LLM; remains the strongest open-source model on SWE-bench Verified. Runs on a single RTX 4090 or Mac with 32GB RAM. Scaffolded over OpenHands or SWE-agent.
