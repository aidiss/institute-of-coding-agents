# Releases — 2026-03-27

## Models

### Claude Sonnet 4.6 / Opus 4.6 (Anthropic)
Current production generation. Sonnet 4.6 is the default model for Claude Code. Opus 4.6 (released February 5) carries a 1M-token context window (beta), 64K default output token limit in Claude Code, and a 14h 30m METR task-completion horizon — the longest tracked for any model. Both power Auto Mode (launched March 24). GA; standard API pricing applies.

### GPT-5.4 (OpenAI) — March 5, GA
Three variants: Standard, Thinking, and Pro. 1M-token context window via API. Consolidates capabilities from GPT-5.3-Codex and separate reasoning variants. Claims 33% fewer factual errors in individual claims vs. GPT-5.2, 18% fewer overall. Record OSWorld and WebArena computer-use benchmark scores (83% on GDPval). Available to Plus, Team, and Pro subscribers; Thinking/Pro variants excluded from free tier.

**GPT-5.4 mini and nano** — Released March 17.

### Gemini 3.1 Flash-Lite (Google) — March 3, Preview
Available via Google AI Studio and Vertex AI. Priced at $0.25/1M input tokens, $1.50/1M output tokens. Claims 2.5× faster time-to-first-token and 45% higher output speed vs. Gemini 2.5 Flash. Adjustable reasoning effort levels.

### Nemotron 3 Super (NVIDIA) — March 11, open weights
Announced at GTC. 120B total parameters, 12B active (hybrid MoE). SWE-Bench Verified: 60.47%. Runs at 2.2× the throughput of comparable open models. NVIDIA Nemotron Open Model License. Targets enterprise coding agent and cybersecurity use cases.

### Qwen 3.5 (Alibaba)
Rolled out across all parameter sizes through February and early March 2026. MoE architecture, 119-language support, Apache 2.0 license. Overtook Llama as the most-downloaded open-source model family in late 2025. Supports video inputs up to two hours.

## Products

### Claude Code Auto Mode (Anthropic) — March 24, Research Preview
Claude decides which actions to take without prompting for confirmation. A safety classifier screens each tool call, blocking mass file deletions, data exfiltration attempts, and prompt injection. Enable via `claude --enable-auto-mode`, toggle with Shift+Tab. Available to Team plan; rolling out to Enterprise and API.

Also shipped simultaneously:
- **Claude Cowork + Computer Use** (March 23): Desktop control (point, click, browse) available to Pro and Max subscribers. Requires local machine to stay awake.
- **Claude Dispatch** (March 17, expanded March 22): Send a task from mobile; Claude works on it on your desktop. Available to Pro ($20/month) and Max ($100–200/month).

**Revenue milestone**: Claude Code surpassed $2.5B annualized revenue in March 2026, up from $1B in early January.

### Cursor v2.6 — March 3, GA
Bundled features: Automations (schedule/event-triggered agents), Cloud Agents with Computer Use, Bugbot Autofix (out of beta — 35%+ merge rate, bug resolution improved from 52% to 76%), MCP Apps, Team Marketplaces, and JetBrains ACP integration. Point releases v2.6.18–v2.6.21 shipped March 10–23. Annual revenue exceeded $2B per Bloomberg, doubling in three months.

### GitHub Copilot Agent Mode — GA (all VS Code users)
Rolled out as part of Microsoft's 50th anniversary. Copilot independently identifies subtasks, executes multi-file changes, runs terminal commands, and self-heals runtime errors. `@copilot` in pull requests can fix failing workflows and push updates back to the branch.

**Copilot Coding Agent REST APIs** — Public preview. Programmatic control of repository access for Copilot agents at the org level.

**Copilot CLI v1.0.10** (March 20): Reduced memory on large file views, multiple concurrent sessions (experimental), `/copy` writes formatted HTML to clipboard on Windows.

**Training data policy change** (effective April 24): Free, Pro, and Pro+ interaction data used for model training by default; opt-out available in Privacy settings. Business and Enterprise excluded.

### Windsurf (Codeium) — ongoing updates
OpenAI acquisition at approximately $3B announced (largest M&A in the cycle). GPT-5.2, GPT-5.1, GPT-5.1-Codex, and Gemini 3 Pro now available in Cascade. BYOK support added for Claude 4 Sonnet, Claude 4 Opus, and Thinking variants. Pricing: Free (25 credits/month), Pro ($15/month), Teams ($30/user/month), Enterprise ($60/user/month).

### Devin v3 API (Cognition AI) — out of beta, March 2026
Includes role-based access control and session attribution. Recent additions: Full Desktop Testing (end-to-end QA via computer use with video recording), Fast Mode (~2x faster at 4× ACU consumption), Devin Review with batch PR comments, and Merge Conflict Detection. PR merge rate improved from 34% to 67% over 2025. Pricing: $20/month Core + $2.25/ACU.

### JetBrains Junie CLI — March 9, Beta
Standalone LLM-agnostic coding agent for terminal, any IDE, and CI/CD. Supports OpenAI, Anthropic, Google, and Grok models. Features planning, subagents, and MCP server support. BYOK. Priced from $10/month (individual) to $60/month (enterprise).

### JetBrains Air — March 9–10, Public Preview (free)
Agentic IDE built on the Fleet codebase. Runs Claude Agent, Gemini CLI, Codex, and Junie CLI side-by-side in a single workspace. Supports Agent Client Protocol (ACP) for vendor-neutral agent-editor communication. macOS now; Linux/Windows coming. Available at air.dev.

### JetBrains Central — March 24, announced (EAP Q2 2026)
Control layer for managing agentic workflows across Air, Junie, and third-party IDEs. Policy enforcement, identity management, cost attribution, and semantic knowledge layer. Design partner program starting Q2.

### Augment Code v0.2.19
Default model switched to GPT-5.4 across Intent, Auggie CLI, and IDE extensions. Completions and Next Edit features being retired March 31 for non-Enterprise plans (shift toward agent-centric workflows). PR sidebar scoped per workspace; custom behavior prompts carry through to delegated specialist agents.

### Diffblue Testing Agent — March 24, GA (new entrant)
Autonomous regression test generator for Java codebases. Works alongside GitHub Copilot, Claude Code, and other enterprise AI coding platforms. Benchmarks: 81% average line coverage across eight real-world Java projects vs. 32% achieved by developer + AI agent iteration. No developer intervention required.

## Open Source

### OpenHands 1.0 — early March 2026
Rebuilt on the new `software-agent-sdk` (composable Python library, MIT licensed). Introduces a Planning Agent with Plan Mode (generates `PLAN.md`) and Code Mode (executes it). **Software Agent SDK** also released separately as a standalone library for external developers.

### OpenCode — new entrant, ~117K GitHub stars by March 2026
Open-source Go-based CLI/TUI coding agent. Supports 75+ models including local models. Privacy-first (no server-side code storage). Available as CLI, desktop app, and VS Code/Cursor extension. Introduced a $10/month hosted tier in March 2026. Competing directly with Claude Code and GitHub Copilot in the open-source segment.

### Cline CLI 2.0
Parallel terminal agents added. VS Code extension now at 5M installs and 58.7K GitHub stars. BYOM pricing model: pay LLM provider rates directly, Cline charges nothing on top.

### Aider — ongoing incremental releases
`reasoning_effort` support for Gemini 2.5 Flash, `--shell-completions` argument, and improved commit message generation. 39K GitHub stars, 4.1M installs, 15B tokens processed per week claimed. No single landmark March release, consistent incremental shipping.
