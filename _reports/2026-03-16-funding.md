### Market Context

- **Claude Code** now leads AI coding tool usage; a survey of ~1,000 software engineers found 95% use AI tools at least weekly — Claude Code ranks first, followed by GitHub Copilot and Cursor.
- AI generates ~30% of Microsoft's code and >25% of Google's code. Meta targets majority-AI code.
- **AGENTS.md** standard adopted by 60,000+ open-source projects; supported by Cursor, GitHub Copilot, Codex, Devin, and VS Code.
- NVIDIA GTC 2026 opened March 16 in San Jose; Jensen Huang previewed the **Feynman architecture** aimed at on-device agentic AI inference.
- Average agentic AI round size in Q4 2025/early 2026: **$155M** (vs. $82M in H1 2025).
- Coding/developer-tool startups collectively raised **>$3B** — the highest-funded agentic AI sub-sector.
