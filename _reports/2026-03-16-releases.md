# Releases — 2026-03-16

## Models

### GPT-5.4 — OpenAI (Mar 5, GA)
Released March 5, GPT-5.4 consolidates reasoning, coding, and computer-use into one model. Notable capabilities:

- **Native computer use** — first general-purpose OpenAI model to interact with software via screenshots, mouse, and keyboard.
- **1,050,000-token context window** with 128K output limit.
- **33% fewer factual errors** than GPT-5.2; 18% fewer full-response errors.
- **Tool search** reduces token costs by 47% in tool-heavy workflows.
- Variants: GPT-5.4 Thinking and GPT-5.4 Pro. GPT-5.1 retired as of March 11.

Pricing: $2.50/$15 per 1M tokens (standard); $30/$180 (Pro). Available via ChatGPT Plus/Pro/Enterprise and the OpenAI API.

Benchmark highlights: 75.0% OSWorld-Verified (vs. human baseline 72.4%), 83.3% ARC-AGI-2 (Pro tier), 83% GDPval knowledge work score.

Cursor surpassed $2B ARR (doubling in three months) and holds ~25% market share among generative AI software buyers.

## Open Source

### OpenHands 1.0.0 — All-Hands-AI (Mar 2026)
Major milestone release built on the new `software-agent-sdk`. Key additions:

- **Planning Agent** (announced Mar 4 product update) — switch between Plan Mode and Code Mode; Plan Mode generates a structured `PLAN.md` outlining implementation steps.
- **Azure DevOps integration** and native tool calling for Gemini 3 Pro Preview.
- Task tracker interface and queued chat messages during runtime connection.
- Raised $18.8M; claims 87% of bug tickets resolved same day on the cloud platform.

MIT-licensed. CLI, local web GUI, and hosted cloud (free tier via Minimax model). Enterprise self-hosting via Kubernetes.
