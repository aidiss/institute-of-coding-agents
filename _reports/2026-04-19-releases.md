# Releases — 2026-04-19

## Models

### Anthropic — Claude

- **Claude Opus 4.7** — released April 16, GA. Available on Anthropic API, Amazon Bedrock, Google Cloud Vertex AI, and Microsoft Foundry. Pricing unchanged from Opus 4.6: $5/M input tokens, $25/M output tokens. Key improvements: stronger coding and vision, higher-resolution image processing, and self-verification — the model can now double-check its own outputs before responding. Ships with automated cybersecurity safeguards that block prohibited use cases; security professionals can join Anthropic's new **Cyber Verification Program** for legitimate red-team and pen-test access. Anthropic publicly acknowledged that Opus 4.7 does not match the internally-held **Mythos** model.
- **Claude Mythos** — released April 7 in restricted preview. Not publicly available; access limited to a handpicked group of tech and cybersecurity companies. Considered Anthropic's current performance ceiling; withheld from GA due to safety review.

### Meta — Llama

- **Llama 4 Scout** — released April 5, open-weight (Apache 2.0). Native multimodal MoE (17B active / 109B total parameters, 16 experts). 10M-token context window — largest of any open-weight model. Available on Hugging Face and llama.com. Powers Meta AI on WhatsApp, Messenger, Instagram, and meta.ai.
- **Llama 4 Maverick** — released April 5, open-weight (Apache 2.0). Same 17B active parameters but 128 experts / 400B total. Outperforms Gemini 2.0 Flash and Mistral 3.1 on coding, reasoning, and image benchmarks. Also powering Meta AI consumer products.

### Google — Gemma

- **Gemma 4** — released April 2, Apache 2.0. Four variants: E2B, E4B, 26B MoE, 31B Dense. Natively multimodal (text + images across all sizes; audio on edge variants). Context windows up to 256K tokens. Built from the same research base as Gemini 3; claims frontier-level performance-per-parameter. Available on Google Cloud and Hugging Face.

### Meta — Muse Spark

- **Muse Spark** — released April 8. Meta's first proprietary (closed-weight) model, separate from the Llama open-weight line. Limited details available at time of reporting.

### Windsurf — Wave 13

Wave 13 (Shipmas Edition) shipped with three major capabilities:

- **SWE-1.5 free tier** — Windsurf's proprietary coding model available at standard throughput at no cost for three months.
- **Arena Mode** — side-by-side blind comparison of two Cascade agents running the same prompt with hidden identities, for model evaluation inside the IDE.
- **Parallel Multi-Agent Sessions** — multiple Cascade agents running concurrently in separate Git worktrees, with multi-pane and multi-tab layouts and a dedicated terminal per agent for more reliable command execution.
- **Plan Mode** — Cascade analyzes the full task scope and creates a structured plan before writing code.

Pricing update (March 19): credit-based model replaced by quota-based; Pro raised from $15 to $20/month; new Max tier at $200/month.

## Open Source

### OpenHands — v1.5.0

- Released in April 2026. Adds **task list** UI, a **planning agent**, and **skill slash menus** — these are in-app task management and planning capabilities.
- Supports switching Git repositories from within existing sessions.
- New integrations: **Bitbucket Datacenter** support; expanded model access.
- Multiple UI fixes and CVE patches.
- GitHub: 69,580 stars (3.7× ahead of SWE-agent at 18,817).
- MIT-licensed; supports Claude, GPT, Gemini, or any LLM backend.

### SWE-agent

- No major release in this period. Remains research-focused (Princeton/Stanford). Primarily an Agent-Computer Interface (ACI) research platform rather than a production tool.
