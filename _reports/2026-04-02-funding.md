## Acquisitions & Exits

**OpenAI acquires Astral** (confirmed, March 19, 2026)
- Astral builds widely used open-source Python tooling: **uv** (package/env manager), **Ruff** (linter/formatter), and **ty** (type checker). Built in Rust; 7M+ monthly downloads; 82,000+ GitHub stars.
- Astral team joins OpenAI's **Codex** team. Financial terms undisclosed. uv, Ruff, and ty will remain open source.
- Context: Codex has seen 3x user growth and 5x usage increase since January 2026, reaching 2M+ weekly active users. OpenAI has made 6 acquisitions in 2026 already.

**Anthropic acquires Bun** (confirmed, early 2026)
- Anthropic's first-ever acquisition. Bun: all-in-one JS runtime, package manager, bundler, and test runner built by Jarred Sumner. 7M monthly downloads; 82,000 GitHub stars; used by Midjourney and Lovable.
- Claude Code ships as a Bun executable — direct infrastructure play. Announced alongside Claude Code reaching **$1B run-rate revenue** (6 months after public GA). Bun remains MIT-licensed open source.

**OpenAI acqui-hires Peter Steinberger / OpenClaw** (confirmed, February 2026)
- Solo developer of open-source AI agent OpenClaw, described as "the fastest-growing open-source AI agent in history," acqui-hired to join OpenAI's agent team.

**Windsurf / Codeium — saga complete** (background, late 2025)
- April 2025: OpenAI announced $3B acquisition of Windsurf — deal collapsed due to Microsoft's contractual rights.
- July 2025: Google acqui-hired Windsurf's CEO, co-founder, and ~40 senior engineers plus secured a technology licensing deal.
- Late 2025: **Cognition AI** (Devin) acquired the remaining Windsurf entity — IP, product, brand, 210 employees, and $82M ARR — for ~$250M. Cognition is merging Windsurf IDE capabilities with Devin for autonomous development workflows. Cognition valuation: $10.2B (Sep 2025), up from $2B (March 2025).

## Trends

- **Code verification is emerging as a category.** Qodo ($70M), Entire ($60M), and Axiom ($200M) all bet that as AI generates more code than humans can review, governance and trust become the next major bottleneck.
- **Vibe coding is now mainstream capital-dense.** The prompt-to-app builder subcategory (Replit, Lovable, Bolt.new, Emergent) attracted ~$800M in combined funding.
- **Model self-sufficiency.** Cursor building Composer 2 on Kimi K2.5 signals coding agent platforms moving to reduce dependency on OpenAI/Anthropic — even while relying on them as infrastructure suppliers.
- **Infrastructure acqui-hires.** Both OpenAI (Astral/uv/Ruff) and Anthropic (Bun) acquired foundational developer tooling rather than agent capabilities, betting that owning the execution environment compounds moat.
