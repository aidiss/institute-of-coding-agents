# Releases — 2026-03-30

## Models

### Grok 4.20 — GA (March 24, 2026)
xAI's Grok 4.20 exited beta on March 24 with three API variants. Key specs:
- **Pricing:** Up to 60% cheaper than Grok 3
- **Context:** 2M-token context window
- **Capabilities:** Multi-agent orchestration, 78% non-hallucination rate on Omniscience benchmark (highest of any tested model)
- **Availability:** xAI API and Grok.com — GA

Note: Grok 5 has slipped from Q1 to Q2 2026.

### DeepSeek V4 (released ~March 3, 2026)
DeepSeek released V4 with a 1 trillion total parameter MoE architecture (32B active per token), 1M-token context window, and native multimodal support. Available via DeepSeek API and open-weight release. Timed to China's Two Sessions political period.

### NVIDIA Nemotron 3 Super — 120B (March 2026)
NVIDIA shipped Nemotron 3 Super, a 120B hybrid Mamba-Transformer MoE model with 12B active parameters per token.
- **Benchmark:** 60.47% on SWE-Bench Verified — #1 among open-weight models for coding
- **Context:** 1M tokens
- **Throughput:** 2.2x vs GPT-OSS-120B
- **Availability:** Open weight

### Qwen 3.5 Small (released March 1, 2026)
Alibaba released four dense models at 0.8B, 2B, 4B, and 9B parameters. All are natively multimodal and Apache 2.0 licensed. The 9B variant scores 81.7% on GPQA Diamond, outperforming GPT-OSS-120B (71.5%).

### Mistral Voxtral TTS (released mid-March 2026)
Mistral's first foray into audio generation: an open-weight text-to-speech model. Available for download and via Mistral API.

### Claude "Mythos" — leaked, not released
Anthropic's next-generation model was accidentally exposed in a public data store. Anthropic confirmed it is in testing and described internally as "by far the most powerful AI we've ever developed," with notes about "unprecedented cybersecurity risks." No release date announced.

## Products

### Cursor — Automations (March 5, 2026)
Cursor launched **Automations**: always-on agents triggered by external events — Slack messages, GitHub PRs, PagerDuty alerts, scheduled times, or arbitrary webhooks. Cursor reports running hundreds of automations per hour internally.

### Cursor — Composer 2 (March 19, 2026)
Cursor shipped **Composer 2**, a new proprietary model/agent system targeting long multi-file coding tasks. Signals a strategic shift: Bloomberg reported Cursor is building its own model to reduce dependency on Anthropic and OpenAI. Cursor has crossed $2B ARR and is reported in talks to raise at a ~$50B valuation.

### Cursor — Self-Hosted Cloud Agents (March 25, 2026)
Code and tool execution can now run entirely within a customer's own network. Targeted at enterprises with data residency requirements.

### Claude Code — Windows PowerShell Preview (March 2026)
Anthropic shipped a Windows PowerShell preview of Claude Code, previously macOS/Linux-only. Also added **computer use** for Pro/Max subscribers (open files, run dev tools, click on-screen without additional setup). Anthropic has shipped 74 releases across Claude Code in the Feb–March window.

### OpenAI Codex — Windows App & v0.116.0 (March 2026)
Codex app launched on Windows (previously macOS-only) on March 4. Version 0.116.0 (March 19) added ChatGPT device-code sign-in, a user prompt hook, and GPT-5.4 Mini as a model option. The **Codex Security agent** (automated threat modeling, sandbox validation, patch generation) also shipped this month.

### OpenAI — Astral Acquisition (announced March 19, 2026)
OpenAI announced the acquisition of Astral, the team behind `uv`, `Ruff`, and `ty` (Python toolchain). The team will work on Codex. Codex now has 2M weekly active users (3x since January).

### GitHub Copilot — Agent Mode GA for JetBrains (March 2026)
Copilot Agent Mode, previously VS Code-only, is now generally available in all JetBrains IDEs. Also shipped: **Agentic Code Review** (project-context-aware PR review with auto-fix PRs), reaching 60M reviews since April 2025 launch.

### Google Antigravity — AgentKit 2.0 (March 2026)
Google's agentic development platform shipped AgentKit 2.0: 16 specialized agents, 40+ domain-specific skills, 11 pre-configured commands. Google AI Studio integration updated with a full-stack coding environment, Firebase backend (auth + database), and expanded framework support (Next.js, React, Angular). SWE-Bench Verified score: 76.2%.
- **Availability:** Public preview, free for individuals

### Microsoft M365 Copilot — Claude Sonnet (announced March 9, 2026)
Microsoft made Claude Sonnet (latest) available to Microsoft 365 Copilot subscribers, expanding the multi-model offering beyond OpenAI models.

## Open Source

### Open-SWE — LangChain AI (March 17–20, 2026)
LangChain's **Open-SWE** is an MIT-licensed async coding agent framework that produces pull requests without human intervention mid-task. Three-agent workflow (Manager → Planner → Programmer + Reviewer), ~15 curated tools, pluggable sandbox (Modal, Daytona, Runloop). Defaults to Claude Opus 4; brings-your-own-LLM. 7,700+ GitHub stars, trending #2 at launch.
- **Repo:** [github.com/langchain-ai/open-swe](https://github.com/langchain-ai/open-swe)
- **Availability:** Open source (MIT); LangSmith Sandboxes hosted option via waitlist
