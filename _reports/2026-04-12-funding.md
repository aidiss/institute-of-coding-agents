## Market Context

**OpenAI's $122B round** (closed March 31) is the defining capital event of the quarter. With Amazon as the anchor investor at $50B, OpenAI achieves infrastructure independence from Microsoft's Azure at scale. Codex — OpenAI's coding agent — is the primary product benefiting, with 2M+ weekly active users and accelerating enterprise adoption. At $852B valuation, OpenAI has surpassed every public software company except Microsoft and Apple by market cap.

**Cursor vs. Claude Code tension:** Anysphere's reportedly imminent ~$60B raise follows news that Claude Code surpassed Cursor as the most-used AI coding tool among engineers (Claude Code 46% vs. Cursor 19% vs. GitHub Copilot 9%, per Anthropic's 2026 Agentic Coding Trends report). Investors appear to be pricing in both the revenue strength ($2B ARR confirmed) and competitive risk.

**Qodo's bet:** As AI agents generate billions of lines of code monthly, Qodo is positioning code *verification* as the next bottleneck — not generation. Enterprise customers include Nvidia, Walmart, Red Hat, Intuit, and Texas Instruments. The $70M Series B (total $120M raised) reflects investor conviction in the "code quality infrastructure" layer.

**Valuation multiples:** Agentic AI–labeled startups are raising seed rounds at ~40% higher valuations than generative AI–labeled peers. Average round size for agentic coding deals in Q1 2026: ~$155M, nearly double the H1 2025 average of $82M.
