# Releases — 2026-04-14

## Models

**Google Gemma 4** (April 2, 2026) — Google's most capable open model family to date, released under a commercially permissive Apache 2.0 license (a meaningful upgrade from the restricted licensing on prior Gemma versions). Four variants: E2B and E4B for edge/mobile, 26B MoE for consumer GPUs, and 31B Dense for cloud deployments. All support up to 256K context, native vision + audio, and 140+ languages. The 31B Dense outperforms models 20× its size on several reasoning benchmarks. Available in Google AI Studio and Google AI Edge Gallery. GA.

**Codestral 2** (April 8, 2026) — Mistral's second-generation code model, 22B dense parameters, released under Apache 2.0 — a notable licensing shift from Mistral's prior code model restrictions. Supports fill-in-the-middle completions. Benchmarks above GPT-4o on HumanEval and MBPP. Available via Mistral API and on Google Vertex AI. GA.

**GLM-5.1** (April 2026) — ZhipuAI's open-weight code model (355B–744B parameters, 32–40B active via MoE), pre-trained on 28.5T tokens. Benchmarks above GPT-5.4 on competitive coding under an MIT license. Available via vLLM and SGLang, and supported as a backend in Cline, OpenCode, and Kilo Code. GA.

## Products

**Claude Code v2.1.101** (April 10, 2026) — Adds `/team-onboarding` command that generates a teammate ramp-up guide from local Claude Code usage history. OS CA certificate store now trusted by default, removing friction for enterprise TLS proxy setups. New interactive Google Vertex AI setup wizard accessible from the login screen. Monitor tool added for streaming events from background scripts. Subprocess sandboxing via PID namespace isolation on Linux (opt-in via `CLAUDE_CODE_SUBPROCESS_ENV_SCRUB`). Fixes Homebrew update prompts and a duplicate-message scroll bug.

**GitHub Copilot CLI — General Availability** (April 2026) — Copilot CLI reached GA this month after a sustained beta. Recent point releases (v1.0.15 April 1, v1.0.17 April 3, through April 10) added Autopilot mode, `--mode`/`--plan` flags, improved MCP tool handling, OpenTelemetry monitoring, custom agent model display-name support, and crash-recovery improvements. GPT-5.4 is the default model; older GPT-5.1 Codex models removed. Enterprise telemetry for org-wide usage tracking included.

**Junie CLI — JetBrains IDE Integration** (April 2026) — JetBrains' LLM-agnostic coding agent Junie CLI (Beta since March) now auto-detects and connects to running JetBrains IDEs without manual setup. JetBrains AI subscribers get full access out of the box; BYOK (Bring Your Own Key) pricing available for other model providers. Supports models from Anthropic, OpenAI, Google, and Grok. Beta.

**Cursor 3.0** (early April 2026) — Major architectural redesign with an Agents Window for running multiple AI agents in parallel, Design Mode for annotating UI elements directly in the browser, a `/worktree` command for isolated Git environment work, and Bugbot for automated code review (~76% resolution rate). GA.

**Windsurf Wave 13** (March–April 2026) — Parallel Multi-Agent Sessions let multiple Cascade instances work simultaneously on separate codebase areas with dedicated terminal profiles. Also adds Git worktree support, multi-pane Cascade, Arena Mode for blind model-quality comparisons, Plan Mode, and the SWE-1.5 model. Pricing moved from credit-based to quota-based; Pro tier raised from $15 to $20/month. GA.

## Open Source

**Google ADK-Python** (April 2026) — Google's multi-agent orchestration framework for Python reached 8,200+ GitHub stars this month. Designed for building and deploying agentic pipelines with tool-use, memory, and cross-agent communication.

**block/goose** (April 2026) — Local-first AI agent with MCP (Model Context Protocol) support, 4,900+ GitHub stars. Runs on-device with full MCP tool discovery and invocation, targeting privacy-sensitive and offline workflows. Apache 2.0.
