# Releases — 2026-04-16

## Models

### Anthropic
- **Claude Sonnet 4.6 & Claude Opus 4.6** — Released February 2026 (GA). Sonnet 4.6 performs at near-Opus quality at Sonnet pricing; Opus 4.6 leads the GDPval-AA Elo benchmark at 1,633 points. Both available via API and Claude products.

### OpenAI
- **GPT-5.4** — Released March 5, 2026 (GA). Achieves record scores on OSWorld-Verified and WebArena Verified computer-use benchmarks; 83% on GDPval knowledge-work test. Bundled into ChatGPT subscriptions.

### Google DeepMind
- **Gemini 3.1 Ultra / Flash-Lite / Flash Live** — Released February–March 2026. Ultra is the flagship reasoning model, Flash-Lite is speed-optimized, Flash Live targets real-time audio. Gemini 3.1 Ultra beats GPT-5.4 and Claude Opus 4.6 on GPQA Diamond (94.3%) and ARC-AGI-2 (77.1%).

### xAI
- **Grok 4.20 Beta 2** — Released March 3, 2026 (beta).

### Mistral
- **Mistral Small 4** — Released March 3, 2026 (GA).

### Meta
- **Llama 4 Maverick** — Available as of April 2026. 400B parameters, 10M-token context window; currently the top open-weight model alongside Google Gemma 4.

## Products

### Cursor (Anysphere)
- **Background Agents** — Shipped as part of 2026 updates, allowing tasks to run asynchronously while the developer continues working. Cursor reached $2B ARR in March 2026.

### GitHub Copilot (Microsoft)
- **Copilot Coding Agent (GA)** — Autonomously works GitHub issues end-to-end: creates branches, writes code, opens pull requests. Multi-model selection now includes GPT-4o, Claude, and Gemini.
- **Jira Integration** — Launched March 2026; connects Copilot to Jira project management in addition to GitHub Issues.

### Windsurf
- **Wave 13** — Shipped with: Parallel Multi-Agent Sessions, Arena Mode (blind model quality testing), Plan Mode (planning separate from code generation), in-IDE local preview, and SWE-1.5 as the new default model (replacing SWE-1).
- **Pricing update** — Pro tier moved to $20/month (matching Cursor); new $200/month Max tier added.

### OpenAI Codex
- **Standalone macOS desktop app** — Launched February 2026. Cloud-based autonomous coding agent also bundled into ChatGPT subscriptions. Includes an open-source CLI.

### Amazon Kiro
- **Spec-driven AI IDE** — Available with free (50 credits), Pro ($20/mo, 1,000 credits), Pro+ ($40/mo, 2,000 credits), and Power ($200/mo, 10,000 credits) tiers. Generates requirements and design docs before implementation; AWS-integrated with configurable steering rules.

### Google Antigravity
- **Agent-first IDE** — Launched November 2025; now maturing in April 2026. Features multi-agent orchestration, built-in browser, and Mission Control. Free preview tier; Pro at $20/month.

## Open Source

### Aider
- Active maintenance continues; 30K+ GitHub stars. Git-integrated, builds internal codebase maps, auto-commits each change with descriptive messages.

### OpenHands (formerly OpenDevin)

### Continue
- 26K+ GitHub stars; only open-source tool with full VS Code + JetBrains support. Last major update April 10, 2026.

### SWE-agent (Princeton/Stanford)
- Remains one of the two leading open-source agentic coding options alongside OpenHands; actively maintained.

## Trends

The dominant pattern across the April 2026 landscape: every major tool is converging on the **agent** model. GitHub Copilot added Agent Mode, Cursor shipped Background Agents, Windsurf's Wave 13 went fully agentic, Codex runs autonomously in sandboxed cloud environments, and Google Antigravity launched agent-first. Competition has shifted from completion quality to full-task execution: long-context retention, planning, tool use, result verification, and PR-ready output.
