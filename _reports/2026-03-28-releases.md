# Releases — 2026-03-28

What shipped this week (March 21–28, 2026) across models, products, and open source.

---

## Models

### Anthropic — Claude Mythos ("Capybara") — Limited Testing
An accidental CMS misconfiguration on March 26–27 exposed ~3,000 internal Anthropic documents, revealing a new model named **Claude Mythos** (internal codename: **Capybara**). Anthropic confirmed the model's existence.
- Described internally as "by far the most powerful AI model we've ever developed," with "dramatically higher scores" on coding, academic reasoning, and cybersecurity benchmarks than Claude Opus 4.6
- A planned fourth product tier above Opus is associated with its launch
- Currently in limited early-access testing; no GA date announced
- Anthropic flagged the model is "currently far ahead of any other AI model in cyber capabilities" — plans to release first to cyber defenders
- Pricing will be "very expensive to serve"; efficiency work ongoing before general release

### Google — Gemini 3.1 Flash-Lite — Preview (Vertex AI / AI Studio)
- 1M token context, 64K max output; $0.25/M input tokens
- Topped 6 of 11 benchmarks vs. GPT-5 mini and Claude 4.5 Haiku
- **Gemini 3.1 Flash Live** (real-time audio-to-audio dialogue) also launched in preview

### Google — Lyria 3 — Preview (API)
- `lyria-3-clip-preview`: 30-second music clips
- `lyria-3-pro-preview`: full-length tracks up to 3 min, 48kHz stereo
- First full-length music generation models from Google available via API

### Mistral — Voxtral TTS — GA (Open Weights + API, March 26)
- 4B-parameter text-to-speech model; first TTS release from Mistral
- Open weights on Hugging Face; API at $0.016/1,000 characters; available in Le Chat and Mistral Studio
- 9 languages; voice cloning from ≥3 seconds of audio; ~70ms latency
- Designed for edge deployment (smartphone/laptop footprint)
- Human evals: exceeds ElevenLabs Flash v2.5 naturalness, matches ElevenLabs v3 quality

### Google — Gemini Drop (March 26, App)
- "Import Memory to Gemini": migrate chat history and memory from ChatGPT, Claude, and Copilot
- Personal Intelligence (cross-Gmail/Photos/YouTube context) made free for US users
- Gemini Live upgraded to 3.1 backbone; autonomous multi-step task handling on Android entered early beta

## Products

### Claude Code — v2.1.86 (March 27, GA)
High-frequency shipping cadence continued throughout the week. Notable in the March 21–27 window:
- Added `X-Claude-Code-Session-Id` header to API requests (useful for proxy aggregation)
- Added `.jj` / `.sl` to VCS exclusion lists (Jujutsu and Sapling support)
- Fixed `--resume` failures on sessions from pre-v2.1.85; fixed Write/Edit/Read on files outside project root
- Fixed MCP step-up authorization flow; fixed memory leak in remote sessions; fixed Python Agent SDK MCP server drop at startup
- `/status` now works mid-response; deep links (`claude-cli://`) open in preferred terminal
- MCP tool descriptions capped at 2KB; token counts ≥1.5M display as "1.5m"
- Non-streaming fallback token cap raised 21k → 64k; timeout raised 120s → 300s
- `/stats` screenshot (Ctrl+S) is now 16× faster
- VSCode extension: rate limit warning banner with usage percentage and reset time
- Availability: GA, included with Claude subscriptions; Max plan at $100/mo

### Windsurf — Pricing Overhaul (Effective ~March 19)
- Replaced credit-based system with quota-based tiers: Free, Pro ($20/mo), Max ($200/mo)
- Pro includes daily/weekly quota refreshes; access to SWE-1, SWE-1.5, Claude Sonnet 4.6, GPT-5, Gemini 3.1 Pro
- Prior $15/mo price advantage over Cursor eliminated; significant developer backlash
- Existing subscribers grandfathered; pre-purchased credits converted to usage quota
- Context: acquired by Cognition (Devin) for ~$250M; Google separately licensed Windsurf tech for $2.4B

### GitHub Copilot — Jira Integration (March 2026, GA)
- Copilot Coding Agent (autonomous PR creation from issues) now accepts Jira ticket assignments directly
- Latest independent benchmark (March 16): Copilot solves 56% of SWE-bench tasks, ahead of Cursor at 52%

### Cursor — JetBrains Support (March 2026, GA)
- Native JetBrains IDE support launched, removing VS Code lock-in
- Company reportedly in preliminary talks for a funding round at ~$50B valuation; crossed $2B ARR

### Devin — Devin Review (Early Release, Free)
- AI code review tool for any public or private GitHub PR
- Groups logically connected changes; categorizes issues by severity (red/yellow/gray)
- Embeds an inline "Ask Devin" session with full codebase context
- Currently free during early release
