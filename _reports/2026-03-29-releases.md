# Releases — 2026-03-29

## Models

### Gemini 3.1 Flash-Lite — GA Preview (March 3, 2026)
Google released Gemini 3.1 Flash-Lite to developers via the Gemini API, Google AI Studio, and Vertex AI. It is the fastest and most cost-efficient model in the Gemini 3 series, targeting high-volume, latency-sensitive workloads (translation, classification, ASR). Key specs:
- **Pricing:** $0.25 / $1.50 per 1M input/output tokens — roughly 1/8th the cost of Gemini 3.1 Pro
- **Context:** 1M tokens
- **Thinking levels:** Minimal, low, medium, or high — configurable per request
- **Throughput:** ~232 tokens/second on the Gemini API

### Claude Sonnet 4.6 / Opus 4.6 (released February 2026, widely available March 2026)
Anthropic's latest generation is now broadly available. Sonnet 4.6 replaced Sonnet 4.5 as the default free tier model on claude.ai. Both models feature a 1M-token context window (beta), improved computer use (leading OSWorld benchmarks), and stronger agent planning. Claude 4.6 scores 80.8% on SWE-Bench, the highest among closed models as of this writing.

### GPT-5.4 (available March 2026)
OpenAI's current flagship model targets business and agentic workflows. Available on ChatGPT Plus ($20/month), Team, and Pro tiers. API pricing: $2.50 / $15 per 1M tokens (standard); $30 / $180 per 1M tokens (GPT-5.4 Pro). Legacy models (GPT-5.2 Thinking) are being deprecated with a June 5, 2026 deadline.

### Grok 4.20 (released February 17, 2026)
xAI's current flagship model uses a four-agent parallel processing architecture: Grok (orchestration), Harper (fact-checking + real-time X data), Benjamin (logic + coding), and Lucas (creative reasoning). Available via xAI API and Grok.com.

## Products

### Cursor — JetBrains ACP Support (March 4, 2026)
Cursor is now available inside IntelliJ IDEA, PyCharm, WebStorm, and other JetBrains IDEs via the **Agent Client Protocol (ACP)**. Requires JetBrains IDE version 2025.3.2+ with the AI Assistant plugin. No JetBrains AI subscription required — available free to all Cursor paid-plan users. Supports model selection across OpenAI, Anthropic, Google, and Cursor's own Composer 2 model.
- **Composer 2 pricing:** $0.50 / $2.50 per 1M input/output tokens (standard); $1.50 / $7.50 (fast variant)
- Cursor has crossed $1B ARR

### GitHub Copilot Coding Agent for Jira — Public Preview (March 5, 2026)
GitHub Copilot can now be assigned Jira issues directly. The agent analyzes issue descriptions and comments, implements changes in an isolated environment, and opens a draft pull request in GitHub — all without requiring the developer to leave Jira. Requires Jira Cloud with Rovo enabled and a GitHub Copilot paid plan.

**March 25, 2026 enhancements:**
- Model selection: specify which model Copilot should use directly from a Jira comment
- Jira ticket number automatically included in PR title and branch name
- Confluence integration via Atlassian MCP server for referencing design documents
- Improved error messages and configuration guidance

Available now on the Atlassian Marketplace (public preview). Also available for GitHub Data Residency customers.

### Windsurf — Pricing Overhaul (March 2026)
Windsurf replaced its variable credit system with fixed quota tiers: **$20, $40, and $200/month**. Tab completions remain unlimited at no credit cost across all plans. The change follows Windsurf's acquisition by Cognition AI (makers of Devin) for ~$250M in December 2025. Wave 13 (December 2025) features — Arena Mode, Plan Mode, and parallel multi-agent sessions — remain available.

## Open Source

### Open-SWE — LangChain AI (March 17, 2026)
LangChain released **Open-SWE**, an open-source, MIT-licensed framework for deploying autonomous coding agents. It captures architectural patterns independently developed at Stripe (Minions), Coinbase (Cloudbot), and Ramp (Inspect). The framework runs fully asynchronously in isolated cloud sandboxes and produces pull requests without human intervention mid-task.

- **Architecture:** Three-agent workflow — Manager (routing), Planner (analysis + human-approved plan), Programmer + Reviewer (execution + PR)
- **Tools:** ~15 curated tools covering shell, web, Git, Linear, and Slack
- **Sandbox providers:** Modal, Daytona, Runloop, LangSmith
- **Default LLM:** Claude Opus 4, pluggable
- **Repo:** [github.com/langchain-ai/open-swe](https://github.com/langchain-ai/open-swe) — 7,700+ stars, trending #2 on GitHub at launch
- **Availability:** Open source (MIT); LangSmith Sandboxes via waitlist
