# Releases — 2026-03-22

What shipped this week across the AI coding agent landscape.

---

## Models

**Xiaomi — MiMo-V2-Pro / "Hunter Alpha"** (revealed March 18, 2026)
- The stealth beta model known as "Hunter Alpha" on OpenRouter was revealed to be **MiMo-V2-Pro**, built by Xiaomi's AI division MiMo under former DeepSeek researcher Luo Fuli.
- Over 1 trillion tokens processed anonymously during beta; topped OpenRouter daily usage charts multiple days in a row.
- Xiaomi's Hong Kong shares surged 5.8% on the announcement.
- Availability: GA; pricing not yet published.

**Google — Gemini 3.1 Flash-Lite** (rolling out, March 2026)
- Efficiency-focused sibling of Gemini 3.1 Pro: 2.5× faster responses, 45% faster output generation.
- Priced at $0.25/M input tokens, making it the cheapest frontier-class multimodal option from a major lab.
- Availability: GA via Google AI / Vertex AI.

**NVIDIA — Nemotron 3 Super** (March 2026)
- 120B parameter hybrid model with 12B active parameters (MoE architecture), delivering 2.2× throughput gains vs. prior generation.
- Targeted at inference efficiency for enterprise deployments, not consumer chat.
- Availability: GA via NVIDIA NIM / API.

**Alibaba — Qwen 3.5** (active as of March 2026)
- Strong multimodal model (text, images, video) positioned for cost-sensitive and global markets.
- Pricing significantly undercuts Western frontier models; exact figures unconfirmed.
- Open weights; available via Hugging Face and Alibaba Cloud.

**DeepSeek — V4 still not released**
- V4 expected to be a 1T-parameter multimodal MoE model; multiple launch windows (mid-Feb, Lunar New Year, early March) have passed without an official release.
- Polymarket prediction market: 50% probability of release by April 15.
- DeepSeek V3.1 remains the current production model.

## Products

**GitHub Copilot — Jira Integration** (March 2026)
- The Copilot Coding Agent now accepts Jira issues as task inputs alongside GitHub issues, extends its autonomous PR workflow into Jira-managed projects.
- 4.7M paid subscribers; 90% of Fortune 100 adoption.
- Pricing: Free (2,000 completions/mo), Pro ($10/mo), Business ($19/user/mo), Enterprise ($39/user/mo).

**Apple — AI-Powered Siri in iOS 26.4** (announced / rolling out March 2026)
- Completely reimagined Siri with on-screen awareness, cross-app integration, and context retention.
- Powered by Google Gemini (1.2T parameters) via Apple's Private Cloud Compute.
- Availability: Preview in iOS 26.4; GA timing dependent on region.

## Open Source

**Open-SWE by LangChain** (released March 17, 2026)
- Asynchronous, cloud-native coding agent built on LangGraph and the LangGraph Platform. MIT license.
- Multi-agent architecture: Manager → Planner → Programmer → Reviewer, each running in isolated Daytona sandboxes.
- Key features: parallel task execution, human-in-the-loop plan approval, auto PR creation with tracking issues, ~15 curated tools (shell, Git, Linear, Slack, web fetch).
- Integrations: Slack (`@openswe`), Linear (`@openswe`), GitHub PR comments (`@openswe`).
- Backstory: captures patterns independently converged on by Stripe (Minions), Ramp (Inspect), and Coinbase (Cloudbot) for internal coding agents.
- Availability: GA — hosted at swe.langchain.com; self-hostable via LangGraph Platform.
- 6,200+ GitHub stars within days of launch.
- Source: [github.com/langchain-ai/open-swe](https://github.com/langchain-ai/open-swe), [blog.langchain.com](https://blog.langchain.com/introducing-open-swe-an-open-source-asynchronous-coding-agent/)

**OpenClaw** (milestone, March 2026)
- Open-source personal AI assistant connecting to WhatsApp, Telegram, Slack, Discord, Signal, iMessage, and Teams; runs entirely on local hardware.
- Crossed **210,000 GitHub stars**; ClawHub skill registry now has 5,700+ community-built skills.
- Availability: Open source; self-hosted.

## Key Themes

- **Stealth benchmarking is now a product launch strategy.** Xiaomi's "Hunter Alpha" was a fully anonymous top-of-leaderboard run before the reveal — a model launch PR move that will likely be replicated.
- **Open-source async agents vs. IDE copilots.** Open-SWE represents a structural alternative to the Cursor/Claude Code paradigm: agents that work in the background, integrate with issue trackers, and deliver completed PRs rather than inline suggestions.
- **Efficiency models filling the gap.** Gemini 3.1 Flash-Lite, GPT-5.4 mini, and Nemotron 3 Super all target the high-volume, latency-sensitive workloads where frontier models are too slow or costly.
- **DeepSeek V4 remains the most anticipated unreleased model.** Every window missed extends the suspense; prediction markets have pushed the expected date to mid-April.
