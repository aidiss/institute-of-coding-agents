# Releases — 2026-04-13

## Models

**Meta Muse Spark** (April 8, 2026) — Meta Superintelligence Labs' first model and Meta's first proprietary (closed) AI release, marking a strategic departure from its open-source Llama lineage. Natively multimodal reasoning model with tool-use, visual chain-of-thought, and multi-agent orchestration support. Available at meta.ai and the Meta AI app (GA). Early reception is mixed — an internal exec acknowledged it trails ChatGPT, Claude, and Gemini in several areas. No public API pricing announced yet.

**Anthropic Claude Mythos** (announced April 7, 2026, no public release) — Anthropic's frontier 10-trillion-parameter system was announced but withheld from public release due to cybersecurity risk. Access restricted to select partners under "Project Glasswing." Not GA; no public pricing.

**Gemini 3.1 Flash Live** (March 26, 2026) — Google's highest-quality real-time audio model, designed for low-latency voice dialogue, video understanding, and tool use in agent pipelines. Available via the Gemini Live API in Google AI Studio and powering Search Live and Gemini Live in 200+ countries (GA). Gemini 3.1 Flash-Lite Preview also surfaced in April for cost-optimized use cases.

**GPT-5.4** (March 5, 2026) — OpenAI's current frontier model with 1M-token context, native computer-use, and integrated coding capabilities from GPT-5.3-Codex. Available to ChatGPT Plus/Team/Pro and via API (GA). Relevant backdrop for the April cycle; no new major OpenAI model shipped this week.

## Products

**Microsoft Agent Framework 1.0** (April 7, 2026) — Production-ready unification of Semantic Kernel and AutoGen into a single open-source SDK. Delivers stable APIs, a long-term support commitment, full MCP support for tool discovery/invocation, and A2A 1.0 cross-framework agent collaboration. Ships with a browser-based DevUI debugger for visualizing agent execution and message flows. GA, open source.

**Claude Code v2.1.97** (April 8, 2026) — Major stability and security pass: 46 changes, 61% bug fixes. New: focus view toggle (`Ctrl+O`) in NOFLICKER mode showing prompt + one-line tool summary + final response; `refreshInterval` status-line setting; `workspace.git_worktree` status-line JSON field; live subagent indicator in `/agents`; Cedar policy file syntax highlighting. Fixed: `--dangerously-skip-permissions` silent downgrade, `--resume` failures, Write/Edit/Read failures outside project root, MCP memory leak. Write tool diff computation 60% faster for large files with tabs/`&`/`$`.

**Claude Code v2.1.96** (April 8, 2026) — Fixes for Bedrock request handling.

**Claude Code v2.1.92** (April 4, 2026) — Adds `forceRemoteSettingsRefresh` policy setting.

**Claude Code v2.1.91** (April 2, 2026) — MCP tool result persistence override.

**Claude Code v2.1.90** (April 1, 2026) — Introduces `/powerup` command for interactive lessons with animated demos.

**Devin Preview Agent Toggle** (March 27, 2026) — Streaming thoughts and faster execution available as opt-in preview; Devin Fast Mode now in the agent picker (~2x faster, 4x ACU cost). Secure mode now available to non-enterprise orgs.

## Open Source

**OpenHands 1.0.0 / V1** (V0 deprecated April 1, 2026) — All My Robot Hands complete architectural transition to V1. The new `software-agent-sdk` separates the agent core from downstream apps, exposes clean Python + REST APIs, and is model-agnostic. V0 is now removed. Recent additions: Azure DevOps integration, native tool calling for `gemini-3-pro-preview`, `/clear` command for V1 conversations. 69K+ GitHub stars. Presented as a poster at MLSys 2026.

**Microsoft Agent Framework 1.0** (April 7, 2026) — Also notable as an open-source release (see Products above). Repo consolidates Semantic Kernel + AutoGen under a single maintained SDK.
