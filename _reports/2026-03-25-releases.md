# Releases — 2026-03-25

What shipped this week across the AI coding agent landscape.

## Models

### Mistral Small 4 (March 16–17)
- **Availability:** GA, Apache 2.0 open weights
- **Specs:** 119B params, MoE (128 experts, 4 active per token, 6B active); 256k context window
- **Notable:** Unifies Magistral (reasoning), Pixtral (multimodal), and Devstral (agentic coding) into one model. Native text + image input. Configurable reasoning effort (fast vs. deep). 40% faster end-to-end latency vs. prior Mistral models. Matches/surpasses GPT-OSS 120B on several benchmarks while producing shorter outputs.

### MiMo-V2 family (March 18)
- **Availability:** Free preview on OpenRouter; MiMo API
- **From:** Xiaomi (team led by former DeepSeek researcher Luo Fuli)
- **Models:**
  - **MiMo-V2-Pro** — Trillion-parameter agent-focused model. 1M context. Ranked #3 globally on PinchBench and ClawEval agentic benchmarks. ~67% cheaper than Claude Sonnet 4.6.
  - **MiMo-V2-Omni** — Multimodal agent model (was "Healer Alpha" in stealth beta).
  - **MiMo-V2-TTS** — Text-to-speech model for expressive voice output.

### NVIDIA Nemotron Speech, RAG & Safety models (March 19)
- **Availability:** GA
- **Nemotron Speech:** ASR model, 10x faster than class competitors, real-time low-latency.
- **Nemotron RAG:** Vision-language embed + rerank for multilingual/multimodal document search.
- **Nemotron Safety:** Expanded Llama Nemotron Content Safety + new Nemotron PII detector.

### GPT-5.4 mini (GA rollout, March 18)
- **Availability:** GA for Free/Go users via the "Thinking" feature; rate-limit fallback for paid subscribers.
- **Note:** GPT-5 Thinking mini retiring as a selectable model within 30 days.

### Context / recently released (informing the week's comparisons)

- **Claude Opus 4.6** (Feb 5, GA) — 64k default / 128k max output; 1M context on Max/Team/Enterprise.
- **GPT-5.4** (March 5, GA) — Three variants (Instant, Thinking, Pro). 1M context. $2.50/$15 per million tokens.

- **NVIDIA Nemotron 3 Super 120B** (March 11–13, NVIDIA Open Model License) — 120B params, 12B active (hybrid MoE). Multi-Token Prediction 3x faster inference. 1M context. #1 on DeepResearch Bench I and II.

## Products

### Claude Code (Anthropic) — March 2026 updates
- **`/loop` scheduled tasks** (GA) — Cron-like recurring task execution within a session. Syntax: `/loop 5m check the deploy`.
- **Voice Mode / push-to-talk** (rolling out) — Activated via `/voice`. Hold spacebar to speak. 20 languages supported (10 new in March).
- **`--channels` MCP message pushes** (research preview) — MCP servers can push messages into a session; permission relay forwards tool-approval prompts to a phone.
- **HTTP hooks** (GA) — POST JSON to a URL, receive JSON back (in addition to shell command hooks).
- **MCP Elicitation support** (GA) — MCP servers can request structured input during execution without interrupting workflow.
- **`--bare` flag** (GA) — Scripted `-p` call mode for automation pipelines.
- **Shared project configs across git worktrees** (GA).
- **Bug fixes:** Auto-updater memory accumulation (was accumulating tens of GB), `PreToolUse` hook security bypass, multi-session re-auth loop, deadlock for large `.claude/skills/` directories.
- **Pricing:** Flat-rate Max plan at $100/month.

### Cursor — March 2026 updates
- **Composer 2** (March 19, GA) — New frontier-level coding model built into Cursor. $0.50/$2.50 per million tokens (standard), $1.50/$7.50 (fast).
- **30+ new plugins** (March 11, GA) — Partners include Atlassian, Datadog, GitLab, Glean, Hugging Face, monday.com, PlanetScale. Available at cursor.com/marketplace.
- **JetBrains integration via ACP** (March 4, GA) — Cursor now available in IntelliJ IDEA, PyCharm, WebStorm, and other JetBrains IDEs. Auth via existing Cursor account.
- **Cursor Automations** (March 5, GA) — Always-on agents triggered by schedules or events from Slack, Linear, GitHub, PagerDuty, or webhooks.
- **BugBot Autofix** (GA) — BugBot spins up a cloud agent to propose fixes directly on PRs; 35%+ of suggestions are being merged.
- **Pricing:** Pro $20/month; Pro+ $60/month; Ultra $200/month.

### Windsurf (Cognition AI) — Wave 14
- **Arena Mode** (GA) — Side-by-side blind model comparison. Two Cascade agents run in parallel on the same prompt; user picks the better result; votes build a community leaderboard. Supports up to 5 models, sync or branched follow-up prompts.
- **Plan Mode** (GA) — Reimagined spec-first planning mode producing a living specification document before code generation. Activated via chat type toggle or typing "megaplan."
- **Pricing:** Pro $15/month (~1,000 prompts/month, unlimited tab completions).

### GitHub Copilot — March 2026 updates
- **Copilot Coding Agent** (GA rollout) — Assign a GitHub issue to Copilot; it writes code, opens a PR, responds to review feedback, self-reviews, and runs security scans.
- **Jira integration** (GA) — Trigger Copilot agents from Jira issues.
- **Full multi-model access** (GA) — Pro+ gives access to Claude Opus 4.6, GPT-5.4, Gemini 3.1 Pro, and Grok Code.

- **Pricing:** Free tier (2,000 completions + 50 premium requests/month); Pro $10/month.

## Open Source

### Open-SWE by LangChain (~March 20–21)
- **License:** MIT
- **Repo:** github.com/langchain-ai/open-swe
- **What:** Asynchronous, cloud-based coding agent framework. Multi-agent architecture: Manager, Planner, Programmer, Reviewer. ~15 curated tools (shell, web fetch, API, Git, Linear, Slack). ## Trends

- **Multi-agent as table stakes.** Parallel agent execution shipped across Grok, Windsurf, Claude Code, Codex CLI, and Devin within roughly the same two-week window. Running parallel agents is no longer a differentiator by this week.
- **MCP proliferating fast.** Model Context Protocol integration now appears in Cursor, Claude Code, OpenHands, and Open-SWE. It is quickly becoming the expected integration standard.
- **Open weights catching up.** Mistral Small 4 (Apache 2.0) and Nemotron 3 Super (NVIDIA Open Model License) are both frontier-adjacent on benchmarks while being fully open weight.
- **IDE reach expanding.** Cursor added JetBrains support March 4, removing a major enterprise adoption objection.
- **Q1 2026 model velocity:** ~255–271+ model releases tracked for the quarter — roughly three new models per day.
