## Acquisitions & Exits

### OpenAI Acquires Astral — Confirmed, Mar 19, 2026
OpenAI will acquire **Astral**, creators of the Python toolchain **uv**, **Ruff**, and **ty** (Ruff alone: ~179M downloads/month). All 30+ employees, including founder/CEO **Charlie Marsh**, join OpenAI's **Codex** division. Financial terms undisclosed.

Rationale: Codex has seen 3× user growth and 5× usage since start of 2026 (2M+ WAU). OpenAI wants Astral to own the full development workflow. uv, Ruff, and ty remain open-source (MIT).

This is OpenAI's sixth acquisition in 2026, nearly matching all of 2025. Other recent deals include Promptfoo (AI testing) and an acqui-hire of OpenClaw creator Peter Steinberger.

### Anthropic Acquires Bun — Confirmed, Dec 2025 *(for context)*
Anthropic's first-ever acquisition: **Bun**, the JavaScript runtime founded by Jarred Sumner. Terms undisclosed. Announced alongside Claude Code reaching $1B ARR run rate. Bun remains open-source (MIT). Claude Code ships as a Bun executable.

### Windsurf / Cognition / Google — Background
In mid-2025, **Google hired Windsurf (Codeium) CEO Varun Mohan, co-founder Douglas Chen, and ~40 employees** for ~$2.4B (~4× invested capital return to investors). **Cognition then acquired the remaining Windsurf business** (IP, product, brand, customer base) for undisclosed terms and is integrating Devin capabilities.

## Notes

- **Cursor's model dependency risk**: Cursor launched **Composer 2** (Mar 19), its first proprietary coding model built atop Chinese open-source Kimi K2.5, priced significantly below Anthropic/OpenAI rates ($0.50/$1.50 per M tokens). Addresses existential risk of depending on model providers it also competes with.
- **Market context**: AI coding tools market reached $4–5B in 2025, projected $12–15B by 2027. GitHub Copilot holds ~42% paid share; Cursor ~18%.
- Broader labor impact: Block, Atlassian, and Salesforce each announced junior engineering headcount reductions citing AI productivity gains.
