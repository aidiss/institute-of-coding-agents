# Releases — 2026-04-11

## Models

### Gemini 3.1 Pro (GA — early April 2026)
Google DeepMind rolled out Gemini 3.1 Pro globally via the Gemini app and API.

- **GPQA Diamond**: 94.3%, ahead of GPT-5.2 (92.4%) and Claude Opus 4.6 (91.3%).
- Focuses on complex reasoning, coding, and data analysis.
- Priced at existing Gemini 1.5 Pro rates; available via Google AI Studio and Vertex AI.

### Gemini 3.1 Flash-Live (GA — early April 2026)
Shipped alongside Gemini 3.1 Pro; optimized for low-latency streaming audio and real-time AI use cases.

### Gemma 4 (Open weights — April 2, 2026)
Google's newest open-weight model family, released under Apache 2.0.

- Four model sizes from edge devices to enterprise.
- 140-language support; natively multimodal.
- Positioned as a local-first backbone for AI coding assistants.
- Weights on Hugging Face and Kaggle Models.

### Meta Muse Spark (Limited rollout — April 8, 2026)
First model from the newly formed **Meta Superintelligence Labs**, marking a notable departure from Meta's Llama open-weight stance — Muse Spark is fully proprietary.

- Natively multimodal with visual chain-of-thought and tool-use.
- Built-in multi-agent orchestration support.
- **AI Index v4.0**: 52 (4th overall).
- Rolling out inside WhatsApp, Instagram, Facebook, Messenger, and Ray-Ban Meta AI glasses.
- No standalone API or pricing announced.

### Mistral Small 4 (`mistral-small-2603`, GA)
Mistral released a hybrid model unifying instruct, reasoning, and coding in a single multimodal package.

- **Context window**: 256K tokens.
- Fully open source; available for fine-tuning or out-of-the-box use via `la-plateforme.io` and Hugging Face.

### Mistral Voxtral TTS (GA)
Mistral's first multilingual text-to-speech model.

- Supports 9 languages with low-latency streaming.
- Custom voice support; API access via Mistral's platform.

### GPT-5.3 Instant Mini (April 9, 2026)
A new OpenAI fallback model replacing GPT-5 Instant Mini for users who hit rate limits on GPT-5.3 Instant.

- Not selectable directly in the model picker; operates as a background rate-limit fallback.
- Accompanied by new Codex usage options on Pro ($20/month) and Plus plans.

## Products

### Cursor 3 (April 2, 2026)
Major interface overhaul from Cursor, representing its largest release since launch.

- New **Agents Window** UI enables parallel agent execution across local, cloud, worktree, and remote SSH environments.
- Agents can be moved between environments mid-session.
- **Design Mode** for precise UI feedback loops.
- **Agent Tabs** for multitasking across concurrent agent sessions.

### Cursor Bugbot — Self-Improvement Update (April 8, 2026)
Bugbot, Cursor's automated PR code-review agent, received a significant capability update.

- **Resolution rate**: ~78% (up from 52% at launch).
- Bugbot can now self-improve its learned rules in real time.
- MCP server support added.
- New **Await tool** lets agents pause on background shell commands or specific output strings.
- Large-file diff rendering significantly optimized.
- Included in Cursor Pro ($20/month) and Pro+ ($60/month); no pricing change.

### Claude Code v2.1.89–v2.1.92 (first week of April 2026)
Four point releases shipped across the week.

- **Amazon Bedrock support** via Mantle integration (opt-in with `CLAUDE_CODE_USE_MANTLE=1`) with an interactive setup wizard.
- Default effort level raised from medium to **high** for API-key, Bedrock/Vertex/Foundry, Team, and Enterprise users.
- **MCP context size limit** raised to 500K tokens.
- Headless permission deferral added.
- Per-model and cache-hit cost breakdown in `/cost`.
- Bugfixes for Homebrew update prompts and multiline cursor behavior.

### GitHub Copilot Cloud Agent (April 1, 2026)
Copilot's cloud agent is no longer limited to pull-request workflows.

- Can now work on a branch without creating one, enabling broader task delegation.
- Part of Copilot Pro ($10/month, includes 300 premium requests) and higher tiers.

### GitHub Copilot CLI v1.0.23 (April 10, 2026)
Latest CLI release in a run of weekly updates.

- Added `--mode`, `--autopilot`, and `--plan` flags.
- Fixed agent hang on first turn.

### GitHub Copilot Agent Mode (GA — March 2026)
Reached general availability on VS Code and JetBrains this cycle.

- Multi-model support including Claude Opus 4.6.
- **Agentic Code Review** gathers full project context and can pass suggestions directly to the agent for automatic fix PRs.

### Windsurf Pricing Overhaul (March 19, 2026)
Codeium replaced its credit system with time-based quotas.

- **Free** / **Pro** ($20/month) / **Max** ($200/month) / **Teams** ($40/user/month).
- Existing subscribers grandfathered through April 2026.
- Change was controversial among heavy users.

### Google Antigravity (GA — growing rapidly in April 2026)
Agent-first IDE launched November 2025; one of the fastest-growing AI dev tools as of April 2026.

- Built on a VS Code fork; powered primarily by Gemini 3.1 Pro and Flash.
- Also supports Claude Sonnet 4.6, Claude Opus 4.6, and GPT-OSS-120B.
- **Manager View** for orchestrating multiple parallel agents across workspaces.
- **SWE-bench Verified**: 76.2%.
- Free for Windows, macOS, and Linux.
- Reached ~6% developer adoption within two months of launch.

### Google Gemini Code Assist (Free tier — March 2026)
Google made the individual tier of Gemini Code Assist completely free with no usage cap announced.

### OpenAI Codex (Updated — April 2026)
Relaunched with GPT-5.4 (see April 10 report) powering repository-scale task handling. Improved performance on structured coding tasks including API implementation and database schema design.

## Open Source

### OpenHands SDK v1.0.0 (GA — ~April 7, 2026)
AllHands released the 1.0.0 stable version of the `software-agent-sdk`, formalizing the OpenHands architecture.

- V0 API deprecated in April 2026.
- **Key architectural shift**: moved from mandatory Docker sandboxing to `LocalWorkspace` as the default (optional sandboxing on demand).
- Event-sourced state model, immutable configuration, typed tool system with MCP integration.
- MIT license (enterprise directory excepted).
- Presented as a poster at MLSys 2026.

### Aider (Ongoing — late March / April 2026)
Continued active development; no major version bump in this window.

- Added support for Claude Sonnet 4 and Opus 4 series with updated model aliases.
- Added Grok-4 support via `xai/grok-4` and `openrouter/x-ai/grok-4`.

### MCP Ecosystem — First Dev Summit (April 2–3, 2026)
The inaugural **MCP Dev Summit** took place in New York City over two days.

- 95+ sessions from protocol maintainers, security researchers, and production deployers.
- MCP now exceeds **97 million monthly SDK downloads** (Python + TypeScript combined).
- Adopted by every major AI provider.
