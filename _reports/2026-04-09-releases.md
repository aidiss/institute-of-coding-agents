# Releases — 2026-04-09

## Models

### Claude Sonnet 5 (GA — April 1, 2026)
Anthropic released `claude-sonnet-5-20260401`, now the default model on claude.ai and available via API at the same $3/$15 per million tokens pricing as Sonnet 4.6.

- **SWE-bench Verified**: 92.4% — a 12-point jump over Opus 4.6 (80.8%), achieved at Sonnet pricing.
- **OSWorld-Verified (computer use)**: 88.3%, above the human expert baseline of 72.4%.
- **GPQA Diamond**: 96.2%, surpassing Gemini 3.1 Pro's previous record of 94.3%.
- 2M token context window now out of beta.
- In Claude Code, developers preferred Sonnet 5 over Sonnet 4.6 roughly 82% of the time in head-to-head use, citing better cross-file context retention and reduced hallucinated completions.

### Google Gemma 4 (GA — April 2, 2026)
Google DeepMind released the Gemma 4 family under a fully permissive **Apache 2.0** license — the first Gemma generation to allow unrestricted commercial use and redistribution.

- **Four sizes**: Effective 2B (E2B), Effective 4B (E4B), 26B Mixture of Experts, and 31B Dense.
- Native multimodal: all sizes process images and video; E2B and E4B also support native audio input.
- Context: 128K (edge models), 256K (larger models).
- The 31B Dense model currently ranks #3 among open models on the Arena AI leaderboard.
- Available on Hugging Face, Ollama, and Kaggle; runs fully offline.

### Alibaba Qwen3.6-Plus (GA — April 2, 2026)
Alibaba released Qwen3.6-Plus, its agentic flagship. Unlike prior Qwen releases, this model is **closed-source** and available only through Alibaba Cloud Model Studio and Qwen Chat.

- 1M token context window by default.
- Targets repository-level coding, multimodal perception (images, documents, video), and autonomous planning/test/iterate loops.
- Compatible with third-party tools including Cline and Claude Code via API.
- Integrated into Alibaba's enterprise platform Wukong.

### Meta Llama 4 Scout & Maverick (GA — April 5, 2026)
Open-weight multimodal MoE models. Maverick (17B active / 400B total, 128 experts, 1M context) and Scout (17B active / 109B total, 16 experts, **10M token context** — largest among open-weight models). Available on Hugging Face, llama.com, and major cloud providers.

### GPT-5.4 (Released March 5, 2026 — broadly available)
OpenAI's current frontier model. Native computer-use capabilities, 1M context via API, record scores on OSWorld-Verified and WebArena Verified. Available as Standard, Thinking, and Pro variants.

### Gemini 3.1 Flash Live (GA — March 26, 2026)
Google's audio-to-audio model for real-time dialogue and voice-first applications.

## Products

### Claude Code — April Updates
Multiple point releases shipping through the first week of April.

- **`/powerup`**: interactive, animated lessons teaching Claude Code features in-session.
- MCP size limit raised to **500K** (previously lower).
- Per-model and cache-hit breakdown added to `/cost` for subscription users.
- `/release-notes` is now an interactive version picker.
- Write tool diff computation **60% faster** on files containing tabs/`&`/`$`.
- Headless permission deferral for CI/scripting environments.
- Bedrock setup wizard added.
- Bug fixes: ctrl+e multiline cursor jump, iTerm2/Ghostty scrolling in fullscreen mode, MCP plugin servers stuck "connecting" on session start, cumulative vs. per-context token hint in `/clear`.
- `@`-file mentions no longer JSON-escape raw string content, reducing token overhead.

### Amazon Kiro (GA — March 2026)
AWS launched Kiro for general availability after a startup-only beta. Agent-first IDE built on VS Code, powered by Claude via Amazon Bedrock, focused on spec-driven development from prototype to production.

- **Pricing**: Free (50 credits/mo), Pro ($20/1,000 credits), Pro+ ($40/2,000 credits), Power ($200/10,000 credits).
- Bundles additional AWS agents for security and DevOps automation.
- Available in AWS GovCloud (US) regions as of February 2026.

### Windsurf — Pricing Restructure (March 2026)
Windsurf replaced its credit system with daily/weekly usage quotas across Free, Pro ($20/mo), Max ($200/mo), and Teams ($40/user/mo) plans. Existing subscribers grandfathered through April 2026.

### GitHub Copilot — Recent Milestones
- **Jira integration** launched March 2026, connecting Copilot to project management beyond GitHub Issues.
- **Premium requests model** (February 2026): Free tier capped at 50/month; Pro at 300; Pro+ at 1,500; additional at $0.04 each.
- Copilot can now be assigned Dependabot alerts for automated remediation, alongside Claude and Codex.

## Open Source

### Gemini CLI v0.36.0 (Stable — April 1, 2026)
Google open-sourced a terminal-based Gemini agent under Apache 2.0. Ships weekly stable releases (Tuesdays at 20:00 UTC) and weekly preview releases (Tuesdays at 23:59 UTC).

- Built-in tools: Google Search grounding, file operations, shell commands, web fetching.
- MCP support for custom integrations.
- 1M token context via Gemini 3.1 models.
- Available via npm; repository: `google-gemini/gemini-cli`.

### Aider — Early April 2026
- Updated default aliases: `sonnet` → claude-sonnet-5, `opus` → claude-opus-4-6.
- `/powerup` lesson support mirrors Claude Code's interactive docs.
- Knight Rider-style spinner animation while waiting for LLM stream to start.
- `--shell-completions` flag generates bash/zsh completion scripts.
- Deepseek models: max output tokens raised to 65,536.

### OpenHands — April 3, 2026 Blog: "The Software Agent Control Plane"
OpenHands published a design post on the Software Agent Control Plane architecture. No version bump in the release dashboard as of April 9; v1.6.0 (released March 30) with Kubernetes support and Planning Mode remains the current stable release.
