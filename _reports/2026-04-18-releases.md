# Releases — 2026-04-18

## Models

### Llama 4 Scout & Maverick — Meta, April 5, 2026 (GA, open-weight)
Meta released two new open-weight multimodal models using a Mixture-of-Experts (MoE) architecture — the first in the Llama family. Both feature 17B active parameters and native support for text, image, and video inputs.

- **Scout**: 16 experts, 10M-token context window. Fits on a single H100. Outperforms Gemma 3 and Gemini 2.0 Flash-Lite on published benchmarks.
- **Maverick**: 128 experts. Beats GPT-4o and Gemini 2.0 Flash across general benchmarks; comparable to DeepSeek v3 on reasoning and coding at less than half the active parameters.
- Available on Hugging Face; powering Meta AI on WhatsApp, Messenger, Instagram, and meta.ai.
- License: open-weight (Meta's custom license)

### Gemma 4 — Google, April 2, 2026 (GA, Apache 2.0)
Google released Gemma 4 in four variants targeting different hardware tiers: E2B (mobile/RPi), E4B, 26B MoE, and 31B Dense. Built from the same research base as Gemini 3.

- Up to 256K context window; native vision and audio processing; 140+ languages.
- Benchmark scores claim to beat models 20× its size; ranked #3 globally on Arena AI among open models.
- Available on Google Cloud and Hugging Face under Apache 2.0 (no commercial restrictions).
- Android AICore Developer Preview also launched alongside the release.

### Mistral Small 4 — Mistral AI, April 2026 (GA, Apache 2.0)
Unified model merging reasoning, vision, and coding into a single endpoint. Free to self-host under Apache 2.0.

### GLM-5.1 — Zhipu AI, April 2026 (GA, MIT)
744B-parameter Mixture-of-Experts model released under MIT license. Reportedly outperforms Claude Opus 4.6 and GPT-5.4 on SWE-Bench Pro.

### GPT-5.4 — OpenAI, March 5, 2026 (GA)
First OpenAI general-purpose model with native computer-use capabilities. 1.05M-token context window. Shows 33% fewer factual errors versus GPT-5.2.

### Gemini 3.1 Pro — Google (GA)
Leads most general benchmarks as of April 2026: 77.1% on ARC-AGI-2, 94.3% on GPQA Diamond.

### Claude Mythos 5 — Anthropic (NOT publicly released)
10-trillion-parameter model announced but withheld from public release due to cybersecurity risk assessment. Available only to select partners under "Project Glasswing."

## Products

### Cursor 3 — Anysphere (GA)
Major version shipped with background cloud agents, local-to-cloud agent handoff, Design Mode, BugBot, Ghost Mode, and multi-pane parallel agents. Anysphere reached $2B ARR in March 2026 and is valued at ~$60B.

- Pricing: Pro $20/mo, Pro+ $60/mo
- Workplace adoption: 18% (tied with Claude Code)

### Windsurf Wave 13 — Cognition AI (GA)
Windsurf (acquired by Cognition AI for $250M in December 2025) shipped Wave 13 with Parallel Multi-Agent Sessions, Arena Mode (blind model comparison), Plan Mode (separate planning from execution), and in-IDE local preview.

- Pricing changed March 19, 2026 from credit-based to quota-based; Pro now $20/mo (matching Cursor).
- Workplace adoption: ~8%

### Claude Code — Anthropic (GA)
Leads SWE-Bench Verified at 80.8% (with Opus 4.6). Terminal-native agent tied with Cursor at 18% workplace adoption.

- Pricing: Max plan $100/mo
- GitHub Copilot still leads overall at 29% workplace adoption.

## Open Source

### OpenHands 1.0 — All Hands AI (GA)
Reached 1.0 milestone. Raised $18.8M. Reports 65K GitHub stars and claims to solve 50%+ of real GitHub issues, with same-day resolution of 87% of bug tickets in testing.

### OpenCode — Anomaly Innovations (Actively growing, early 2026)
Terminal-based open source coding agent that gained 18,000 stars in two weeks in early 2026 after developers discovered Claude Max subscription routing support. Now at 140K+ stars, 6.5M monthly developers, 850+ contributors. Supports 75+ models; available as CLI, desktop app, and IDE extension (VS Code, Cursor, JetBrains, Zed, Neovim).

### Aider (Stable)
40K+ stars, 4.1M installs. Deep Git integration with auto-commits. Considered the most mature terminal-based agent.

### Continue.dev (Stable)
26K stars. Only major open source agent with full VS Code + JetBrains support. IDE-native approach.
