**HumanEval:** Saturated at ~99%. Retired as a meaningful discriminator.

### Analyst Note

The SWE-bench Verified era is over. OpenAI's Feb 23 retirement announcement — backed by evidence that 59%+ of remaining tasks are flawed or contaminated and that frontier models reproduce verbatim gold patches — is a watershed moment. The 57-point gap between Verified (80%) and Pro (23%) is the starkest demonstration yet of Goodhart's Law in AI evaluation: the benchmark became the target, and ceased to be a good measure. Community migration is underway toward SWE-bench Pro, SWE-bench Live (dynamically updated), and private benchmarks. Expect leaderboard narrative chaos for 60–90 days as the authoritative standard is contested. Trend direction: **capability accelerating; evaluation credibility stagnating**.

## Emerging Threats
_As of 2026-03-05_

### High Priority — Watch Immediately

**Replit — $400M raise, $9B valuation** _(verified)_
A 3x valuation jump in months. Replit Agent (multi-file, full-stack, database-to-deployment) is now funded to compete head-on with GitHub Copilot and Cursor at enterprise scale. The funding signals that the platform layer above models is consolidating fast — distribution advantage may matter more than raw benchmark scores within 90 days.

**Grok 4.20 (xAI) — four-agent architecture** _(verified, announced Feb 17)_
Architecturally distinct from all current leaders; not just a bigger model. Full release + open API expected Q2 2026. If it performs as pitched, it could reorder rankings significantly within 60 days.

**DeepSeek V4 — 1T parameters, native multimodal** _(expected ~Mar 3; unconfirmed as of report date)_
DeepSeek has surprised the market before. At 1T params with multimodal capability, arrival could shift open-weight and commodity-pricing dynamics immediately.

**Entire (Thomas Dohmke, ex-GitHub CEO) — $60M seed, $300M valuation** _(verified, announced Feb 10)_
Largest developer tools seed in history. Investors include Microsoft's M12 — notable strategic signal. Product: "Checkpoints," an open-source CLI embedding prompts, reasoning, and decisions alongside Git commits for AI-written code. Not competing with Cursor/Claude Code — building the governance/audit layer above them. If this format achieves adoption like `.gitignore`, it becomes infrastructure that everything else plugs into. Watch for GitHub acquisition or clone within 90 days.

### Medium Priority — Track Weekly

**Reflection AI — $105M Series A, ~$555M valuation** _(verified)_
RL-trained agents that write, debug, test, and deploy code end-to-end. If product lands, could disrupt the "human in the loop" assumption underlying most current tools.

**MiniMax M2.5 — open-weight, #3 globally** _(verified)_
80.2% SWE-bench Verified as an open-weight model. Lowers the barrier for startups and self-hosted deployments to access near-frontier coding capability — compresses commercial moats.

**Kilo Code — $8M seed, 750K downloads, #1 on OpenRouter** _(verified)_
Open-source coding agent processing 6 trillion tokens/month. Fast community traction; positioned to become the default open-source layer under proprietary models.

**Gemini 3.1 Pro (Google, Feb 19)** _(verified)_
Leads 13/16 tracked benchmarks; 77.1% ARC-AGI-2; 1M-token context. Google's distribution integration (Workspace, Android, Cloud) is now backed by frontier-grade coding capability.

**SolveAI — $50M Series A (Google Ventures)** _(verified, Feb 25)_
Ex-Palantir team; differentiates on deep company-specific context ingestion. Enterprise coding play.

**Ai2 SERA — open-source agent research** _(verified)_
Allen Institute open-sourced full coding agent stack (models + training). Built largely by one researcher; replication cost now in hundreds of dollars. Could democratize agent research and produce a sudden cluster of capable open-source competitors.

### Low Priority — Noted

- **GPT-5.3 Codex (OpenAI, Feb 5):** Iterative specialist release for terminal-heavy workflows. No architectural surprise.
- **Claude Sonnet 4.6 (Anthropic, Feb 17):** Near-Opus performance at Sonnet pricing; default in GitHub Copilot's coding agent. Anthropic consolidating distribution.
- **Emergent (YC-backed):** $100M raised, $50M ARR in 7 months, 5M+ users, 6M+ apps built. Fastest ARR velocity in the cohort.
- **Sled:** Mobile voice interface for Claude Code/Codex/Gemini CLI. Solves agentic coding interruption friction. Early stage; worth monitoring adoption.
- **Cursor:** $29B valuation — established leader, no new developments this cycle.

### Analyst Note

Two disruption vectors to watch:

1. **Architecture:** Grok 4.20's multi-agent design and DeepSeek V4's scale represent bets that the next capability step comes from systems-level innovation rather than more pretraining. If either delivers in Q2, it fractures the current SOTA cluster.

2. **Governance layer:** As agents produce the majority of committed code, enterprises will demand auditability. Entire is the first well-funded bet squarely on this problem. Microsoft's M12 participation makes GitHub alignment plausible. The governance layer could become the chokepoint that determines which coding agents enterprises are permitted to use — more durable than any benchmark advantage.

## Overall Assessment

**ACCELERATING — with compounding uncertainty.**

Q1 2026 is a landmark period across every dimension: 255+ model releases tracked, SWE-bench Verified collapsing as an evaluation standard, record capital deployment, and architecturally novel approaches entering the arena. The Chinese model cohort has arrived at the frontier. The governance problem above agents is now investable.

The hard capability ceiling (SWE-bench Pro ~23%) is barely dented — all the headline "80% agents" are still far from solving complex real-world software. The next 60–90 days will test whether Grok 4.20 and DeepSeek V4 deliver genuine architectural breakthroughs, and whether the governance layer (Entire) defines the enterprise adoption bottleneck before incumbents react.
