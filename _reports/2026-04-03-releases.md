# Releases — 2026-04-03

What shipped across the AI coding agent landscape in late March through early April 2026.

---

## Models

### Anthropic

**Claude Sonnet 4.6** — GA, released ~February 17, 2026
Near-Opus performance at Sonnet pricing. Preferred over the previous Sonnet 70% of the time in Claude Code testing. Leads the GDPval-AA Elo benchmark at 1,633 points. Available via API and claude.ai.

**Claude Opus 4.6** — GA, released February 5, 2026
Flagship model with "agent teams" capability and native PowerPoint integration. Supported in OpenHands 1.5.0. Priced at Opus tier.

**Claude Mythos ("Capybara" tier)** — Limited early access only, no GA date set
Revealed via a leaked blog draft on March 26, 2026. Described internally as a "step change" beyond Opus — a new tier above Opus, not a point release. Focused on autonomous multi-step agentic reasoning. Currently in early access with select cybersecurity partners. No public release date confirmed.

### OpenAI

**GPT-5.4** — GA, released March 5, 2026
Released with Standard, Thinking, and Pro variants. Key specs: native computer use at 75% on OSWorld-Verified (above the 72% human average), 1M token context window, 33% fewer factual errors vs GPT-5.2, configurable reasoning effort (5 levels). Pricing: $2.50/MTok input for standard.

**GPT-5.4 Mini and Nano** — GA, released March 17, 2026
Smaller, cost-optimized variants bundled into ChatGPT subscriptions.

### Google DeepMind

**Gemini 3.1 Pro** — Preview, released February 19 with wider rollout in March 2026
Leads 13 of 16 major benchmarks; ties GPT-5.4 Pro on the Artificial Analysis Intelligence Index at roughly one-third of the API cost. 77.1% on ARC-AGI-2. 1M token context window. Available via Gemini API, Vertex AI, and Gemini app (Pro/Ultra plans). Still listed as preview.

**Gemini 3.1 Flash-Lite** — Preview, launched March 3, 2026
Speed-optimized, low-cost variant.

### Mistral AI

**Mistral Small 4** — GA, Apache 2.0, released March 16, 2026
119B-parameter MoE model with 6B active experts. Consolidates reasoning (Magistral), multimodal (Pixtral), and agentic coding (Devstral) into one unified model. 256k context window, configurable reasoning effort. Available for fine-tuning and self-hosting.

**Voxtral TTS** — GA via API, released ~March 2026
Mistral's first multilingual text-to-speech model. Supports 9 languages, low-latency streaming, and custom voices.

## Products

### Claude Code (Anthropic)
Multiple updates shipped across March 2026. Bundled with Pro and Max subscriptions.

- **March 23**: Computer use added — Claude can open files, run dev tools, click/navigate the screen; no setup required.
- **Cowork mode**: Collaborative multi-agent sessions.
- PowerShell tool preview for Windows.
- `--bare` flag for scripted/headless calls.
- `--channels` permission relay for forwarding approval prompts to phone.
- PermissionDenied hook with retry support.
- "defer" permission decision in PreToolUse hooks for headless session resumption.
- Named subagents in @ mentions.
- System-prompt caching with ToolSearch and MCP tools.
- Flicker-free alt-screen rendering via `CLAUDE_CODE_NO_FLICKER=1`.

### GitHub Copilot
- **Jira integration** — GA, launched March 2026.
- **Agent Mode** added — autonomous multi-file task execution.
- 4.7M paid subscribers, 20M total users, 90% Fortune 100 adoption as of early 2026.

### Cursor
- **Background Agents**: Cloud agents spin up sandboxes, clone repos, and work on branches independently.
- **Subagents**: Parallel subagents explore different parts of the codebase simultaneously.
- **Debug Mode**: Instruments code and uses real execution data rather than pattern matching.
- **Plan Mode**: Clarifying questions before execution to reduce wasted tokens.

### Windsurf
March 2026 updates:
- New quota-based billing replacing variable credits: tiers at $20, $40, and $200/month.
- GPT-5.4 and Claude Opus 4.6 now selectable as model choices.
- New model picker with family grouping, hovercard variants, and pinning.
- VSCode base upgraded to 1.110.
- Ranks #1 in LogRocket AI Dev Tool Power Rankings (February 2026).

### Devin (Cognition)
- Pricing dramatically cut: from $500-only to a **$20/month Core plan** plus pay-as-you-go ACU pricing at **$2.25/ACU**.

### Augment Code
- **Auggie CLI 0.20.0** (March 19): @ Context Picker, Marketplace Auto-Update, MCP OAuth Paste Mode, configurable Effort Levels, parallel tool execution, notification bell for agent prompts.
- **Intent 0.2.24** (March 11): Pinned projects, edit button for suggested answers.
- **Intent 0.2.25** (March 12): Figma MCP as one-click install, multi-provider support for all agent types, OAuth-capable embedded browser.
