### Analyst Note

The benchmark landscape just bifurcated. SWE-bench Verified served its purpose — it drove agents from ~20% to 80% in roughly two years — but OpenAI's finding that frontier models can reproduce verbatim gold patches exposes fundamental data integrity problems. The community is migrating to SWE-bench Pro and LiveCodeBench as the authoritative difficulty floors. At ~23% on Pro, all the "80% agents" are still nowhere near solving hard real-world issues. The Chinese model cohort (MiniMax, GLM, Kimi, DeepSeek) now competes directly at the frontier on standard evals — the field is no longer a two-lab race.

## Emerging Threats
_As of 2026-03-05_

### High Priority — Watch Immediately

**Entire** (Thomas Dohmke, ex-GitHub CEO)
- Announced 2026-02-10. $60M seed at $300M valuation — largest seed in developer tools history. Investors: Felicis, Madrona, M12 (Microsoft), Basis Set.
- Product: "Checkpoints" — open-source CLI that embeds prompts, reasoning, and decisions alongside Git commits for AI-written code. Supports Claude Code and Gemini CLI on launch.
- Thesis: As agents write more code than humans can review, the governance/audit layer becomes critical infrastructure. Not competing with Cursor/Claude Code — building above them.
- Risk: Could become the de facto standard for AI code provenance, locking in integrations before others react. Microsoft's M12 participation is notable — potential GitHub alignment.

**Bolt.new** (acquired $400M, Jan 2026 — valuation $3B+)
- Browser-native full-stack agent now very well-capitalized. Targeting non-technical users building production apps. Volume play with potential to commoditize lower-tier coding work.

### Medium Priority — Track Weekly

**SolveAI** (London, ex-Palantir)
- $50M raised (Google Ventures Series A). Differentiator: deeply company-specific context. Enterprise coding with proprietary codebase ingestion.
- [Fortune, 2026-02-25](https://fortune.com/2026/02/25/exclusive-solveai-eight-months-raises-50-million-ai-coding-tool-race/)

**Emergent** (YC-backed)
- $100M raised. $50M ARR in 7 months, 5M+ users, 6M+ apps built. Autonomous generation-to-deployment pipeline. Fastest-growing in the cohort by ARR velocity.

**Ai2 SERA** (open-source)
- Allen Institute released full open-source coding agent — models, training recipes, Claude Code integration — built largely by a single researcher. Bringing replication cost down to hundreds of dollars. Could democratize agent research and accelerate open-source competition.

### Low Priority — Noted

- **OpenCode**: 100K+ GitHub stars, 2.5M monthly developers. Established OSS agent — not new but continuing to grow.
- **Magic**: $465M raised, 100M-token context window coding models. No recent news.
- **Cursor**: $29B valuation — established leader, no new developments this cycle.

### Analyst Note

The most likely near-term disruption is the governance/audit layer. As agents produce the majority of committed code, enterprises will demand auditability — who prompted what, when, why. Entire is the first well-funded bet squarely on this problem. If the "Checkpoints" format achieves adoption the way `.gitignore` or `package.json` did, it becomes infrastructure that everything else plugs into. Watch whether GitHub itself (Microsoft) acquires or clones this within 90 days. Secondary disruption: Ai2's SERA democratizing agent research could produce a sudden cluster of capable open-source agents, compressing the moat of commercial players.

## Overall Assessment

**The field is accelerating — but bifurcating.**

The top of the stack (benchmark performance, funding, agent capability) is moving fast. SWE-bench Verified crossed 80%, a threshold that seemed distant 18 months ago. Funding is at historic scale — a single developer tools seed round now clears $60M. The Chinese model cohort has arrived at the frontier.

At the same time, the hard problems (SWE-bench Pro ~23%) are barely dented, governance of agent-written code is unsolved, and the "productivity paradox" (high-velocity code generation creating technical debt) is emerging as the next bottleneck. The next 60–90 days will test whether the governance layer (Entire, others) and harder benchmarks (Pro, LiveCodeBench) redefine what "good" looks like in this space.
