# Releases — 2026-04-06

What shipped this week across models, products, and open source.

---

## Models

**Google Gemma 4** — Released April 2, 2026 (GA, Apache 2.0)
Four open-weight model sizes released simultaneously: E2B and E4B (edge/mobile), 26B MoE (server workloads), and 31B Dense (compute-intensive). Apache 2.0 licensed — a first for the Gemma line. Key specs: 256K context window, native vision and audio, 140+ language support, 4× faster and 60% lower power draw vs. prior Gemma generation. Weights available on Hugging Face, Kaggle, and Ollama. Built on the same research base as Gemini 3.

**Alibaba Qwen 3.6-Plus** — Released April 2, 2026 (GA, proprietary)
1M token default context window with native multimodal support (can generate front-end code from screenshots and design drafts). Alibaba claims parity with Claude Opus 4.5 on SWE-bench and Terminal-Bench 2.0; GPQA score reported at 0.904. Available via Alibaba Cloud Model Studio and Qwen Chat; free preview on OpenRouter.

**Zhipu AI GLM-5V-Turbo** — Released April 2, 2026
Multimodal vision model from Chinese lab Zhipu AI optimized for high-throughput inference. Limited public benchmark data at launch.

## Products

**Cursor 3** — Released April 2, 2026 (GA)
Major product revision focused on multi-agent orchestration:
- **Agents Window**: manage multiple simultaneous local, cloud, remote SSH, and git-worktree agents from a unified sidebar
- **Design Mode**: annotate and target UI elements directly in-browser for precise agent direction
- **Agent Tabs**: view multiple agent chats side-by-side or in a grid layout
- `/worktree` command: creates isolated git worktrees so parallel agents don't collide
- **Composer 2** model bundled with high usage limits
No pricing change announced (Pro $20/mo, Pro+ $60/mo, Ultra $200/mo).

**Google Gemini API — Flex and Priority Tiers** — Released April 3, 2026 (GA)
Two new service tiers giving developers explicit cost/reliability/latency control:
- **Flex tier**: 50% cheaper than standard; latency-tolerant (1–15 min response window); synchronous interface
- **Priority tier**: 75–100% premium over standard; routes to high-criticality compute; per-token billing; available to Tier 2/3 API users
No code changes required to existing endpoints to adopt.

**Microsoft MAI-Transcribe-1, MAI-Voice-1, MAI-Image-2** — Released April 2, 2026 (GA)
Three in-house models from Microsoft's new superintelligence team, signaling a deliberate move to reduce dependence on OpenAI:
- **MAI-Transcribe-1**: Speech-to-text across top 25 languages; 2.5× faster batch transcription vs. Azure Fast. Pricing: $0.36/hour of transcribed speech.
- **MAI-Voice-1**: Text-to-speech with emotional range and speaker identity preservation; generates 60 seconds of audio in under 1 second on a single GPU. Pricing: $22/1M characters.
- **MAI-Image-2**: Text-to-image; debuted at #3 on Arena.ai image leaderboard. Pricing not disclosed.
Available via Microsoft Foundry and MAI Playground.

**Google AI Edge Gallery (Gemma 4 update)** — Released April 6, 2026 (GA, free)
Official iOS/Android app updated to support Gemma 4 E2B and E4B fully on-device. New additions: Thinking Mode (step-by-step reasoning display) and Agent Skills (Wikipedia fact-grounding, maps integration, visual summaries).

Represents a significant product direction shift toward enterprise cloud deployment. Main repo at ~69,500 GitHub stars.
