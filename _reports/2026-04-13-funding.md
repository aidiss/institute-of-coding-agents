## Acquisitions & Exits

### OpenAI — Astral (March 19, 2026) ✓ Confirmed

OpenAI acquired **Astral**, a small startup behind widely-used open-source Python developer tooling (including `uv`, the Rust-based Python package manager, and `ruff`, a fast Python linter). The entire Astral team joined OpenAI to accelerate its **Codex** coding agent. No financial terms disclosed. Strategic intent: giving Codex deep integration with the Python developer toolchain layer, mirroring Anthropic's earlier acquisition of the Bun JavaScript runtime for Claude Code.

**Context:** Codex crossed **2 million weekly active users** at time of deal, with a 3× increase in user growth since January 2026.

---

### Anthropic — Subscription Access Restriction (April 4, 2026) ✓ Confirmed

Anthropic announced that **Claude subscriptions no longer grant API access for third-party tools** — affecting Cline, Cursor, Windsurf, OpenClaw, and any non-Anthropic harness. Automation users were moved to direct API billing. This is not a traditional acquisition but functions as a strategic moat: subscription economics remain within Anthropic's own toolchain (Claude Code), while third-party integrations shift to metered API costs. Widely interpreted as Anthropic consolidating revenue attribution for Claude Code's growth.

## Context & Trends

**Subscription vs. API bifurcation:** Anthropic's April 4 policy change codifies a split that has been building for months — AI labs want to own end-user relationships, not wholesale model access. Cursor and Windsurf now face higher input costs and must price accordingly.

**Toolchain acquisitions as strategy:** Both OpenAI (Astral) and Anthropic (Bun, earlier) are acquiring developer tooling startups rather than pure model companies. The thesis: owning the execution and packaging layer gives coding agents runtime-level advantages (performance, security, guardrails) that pure prompt engineering cannot replicate.

**Verification as the next frontier:** Qodo's $70M Series B reflects investor conviction that *correctness* — not generation speed — is the next competitive moat. As AI-written code volumes scale, automated verification becomes infrastructure.

**Cursor $50B valuation race:** If the reported ~$5B raise closes at a ~$50B valuation, it would represent a 70%+ premium over the November 2025 Series D in under five months, making Cursor one of the fastest-appreciating private software companies on record.
