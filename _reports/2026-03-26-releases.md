# Releases — 2026-03-26

## Models

### Claude Sonnet 4.6 / Opus 4.6 (Anthropic)
Current production generation. Sonnet 4.6 leads the GDPval-AA Elo benchmark at 1,633 points. Opus 4.6 offers a 1M-token context window (beta) and 128K output. Both are required for Claude Code Auto Mode (launched March 24). GA; Sonnet priced at standard Sonnet-tier API rates, Pro plan at $17/month.

### GPT-5.4 (OpenAI) — March 5, GA
Three variants: Standard, Thinking, and Pro. Up to 1.05M context tokens. Includes Tool Search for agentic cost/latency reduction. SWE-Bench Pro score: 57.7%. Bundled into ChatGPT Plus/Pro. Note: legacy deep research mode deprecated March 26.

### Gemini 3.1 Flash-Lite (Google) — GA
Enterprise-scale release at $0.25/M input tokens. 2.5× faster TTFT vs. Flash 2.5. Adds "adjustable thinking levels" for modulating reasoning effort.

### Nemotron 3 Super (NVIDIA) — March 11, open weights
120B total parameters, 12B active (hybrid MoE). Licensed under NVIDIA Nemotron Open Model License. SWE-Bench Verified: 60.47% (OpenHands scaffold) vs. GPT-OSS-120B's 41.90%. 91.75% on RULER at 1M tokens. 2.2× higher throughput than GPT-OSS-120B. Targets multi-agent software dev and cybersecurity.

## Products

### Claude Code Auto Mode (Anthropic) — March 24, Research Preview
The week's highest-impact coding-agent launch. A classifier evaluates every tool call before execution, screening for mass file deletions, data exfiltration, and prompt injection. Safe actions proceed automatically; risky ones are blocked. Enable via `claude --enable-auto-mode`, toggle with Shift+Tab. Available on Team plan; rolling out to Enterprise and API users. Requires Claude Sonnet 4.6 or Opus 4.6. Sandboxed environments recommended.

Also shipped simultaneously:
- **Claude Cowork Computer Use**: direct keyboard-and-mouse control of macOS desktops.
- **Claude Code Channels**: control Claude Code via Discord and Telegram.

### IntelliJ IDEA 2026.1 (JetBrains) — March 25, GA
Java 26 day-one support, Kotlin 2.3.20, C/C++ in IDEA for multi-language projects, JavaScript without Ultimate subscription. Supports Codex/Cursor/ACP-compatible agents via built-in hooks. Includes JetBrains Console for team AI usage analytics. Over 1,000 bugs fixed. Code With Me unbundled and deprecated (relay infrastructure off Q1 2027).

### JetBrains Air — ~March 19, Public Preview (free)
Agentic development environment built on the Fleet IDE codebase. Delegates tasks to multiple concurrent AI agents (OpenAI Codex, Claude Agent, Gemini CLI, Junie). Tasks defined with symbol-level context (line, commit, class, method). Currently free.

### JetBrains Central — March 24, announced (EAP Q2 2026)
Governance and orchestration layer connecting JetBrains IDEs, third-party IDEs, CLIs, and AI agents. Provides policy enforcement, identity management, cost attribution, cloud runtimes, and a semantic knowledge layer. Not GA — design partners can apply for Q2 EAP.

### Cursor Composer 2 — March 19, GA
Benchmarks: 61.3 CursorBench, 61.7 Terminal-Bench 2.0, 73.7 SWE-bench Multilingual (vs. Composer 1.5's 44.2/47.9/65.9). 200K context. Pricing: $0.50/M input (standard), $1.50/M (fast, now default). Controversy: revealed to be built ~25% on Moonshot AI's Kimi K2.5 (open-source) plus ~75% proprietary continued pretraining/RL; Cursor did not disclose the base model at launch.

### GitHub Copilot Jira Integration — March 5, Public Preview
Assign a Jira issue to Copilot from the assignee dropdown; the agent autonomously opens a draft PR in GitHub and posts updates back in Jira. Requires Jira Cloud + Rovo, a paid Copilot plan, and the GitHub Copilot for Jira app from the Atlassian Marketplace.

### Google AI Studio Full-Stack Vibe Coding — March 18, GA
Antigravity coding agent integrates natively with Firebase in AI Studio. Detects when an app needs a database or auth, auto-provisions Cloud Firestore and Firebase Authentication, and deploys to React/Angular/Next.js. Installs dependencies (Framer Motion, Shadcn, etc.) automatically.

## Open Source

### Open-SWE (LangChain) — March 17
First open-source framework for async coding agents running in cloud sandboxes. Accepts mid-run feedback; integrates with Slack, Linear, and GitHub. Three-agent architecture on LangGraph: Manager, Planner (pauses for human approval), and Executor. Targets multi-file refactors, test creation, dependency updates, feature scaffolding, and bug-fix from GitHub issues. 7,700+ stars at launch.

### MolmoWeb (AI2) — March 24, Apache 2.0
Open visual web agent operating from screenshots only (no page source). Two sizes: 4B and 8B parameters. Weights, training data, code, and evals all Apache 2.0. Performance: 8B scores 78.2% on WebVoyager (top open-weight), 94.7% pass@4 with test-time scaling. Ships with MolmoWebMix: 30K human-completed web tasks, 590K+ actions across 1,100 websites — the largest publicly released collection of its kind.

### Java 26 — March 17, GA
10 JEPs delivered, 5 final. No new stable language features; meaningful performance improvements and library additions. IntelliJ IDEA 2026.1 ships with day-one support.
