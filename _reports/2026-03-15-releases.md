### Claude Code v2.1.76 — Anthropic (Mar 14, GA)
Released yesterday. Key additions:

- **MCP elicitation support** — MCP servers can now request structured input mid-task via an interactive dialog (form fields or browser URL); new `Elicitation` and `ElicitationResult` hooks let you intercept and override responses before they're sent back.
- **`-n / --name` CLI flag** — sets a display name for the session at startup.
- **`worktree.sparsePaths` setting** — for `claude --worktree` in large monorepos, checks out only the specified directories via git sparse-checkout.
- **`PostCompact` hook** — fires after context compaction completes.
- **`/effort` slash command** — sets model effort level inline.
